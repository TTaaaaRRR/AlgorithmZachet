### 1. –ü–æ–Ω—è—Ç–∏–µ –∏ —Å–≤–æ–π—Å—Ç–≤–∞ –∞–ª–≥–æ—Ä–∏—Ç–º–∞

**–ê–ª–≥–æ—Ä–∏—Ç–º** ‚Äî —ç—Ç–æ –∫–æ–Ω–µ—á–Ω–∞—è –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —á—ë—Ç–∫–æ –æ–ø—Ä–µ–¥–µ–ª—ë–Ω–Ω—ã—Ö –¥–µ–π—Å—Ç–≤–∏–π (–∏–Ω—Å—Ç—Ä—É–∫—Ü–∏–π) –¥–ª—è —Ä–µ—à–µ–Ω–∏—è –∫–æ–Ω–∫—Ä–µ—Ç–Ω–æ–π –∑–∞–¥–∞—á–∏.

* **–ü—Ä–∏–º–µ—Ä –∏–∑ –∂–∏–∑–Ω–∏:** –†–µ—Ü–µ–ø—Ç —Ç–æ—Ä—Ç–∞.
* **–ü—Ä–æ—Å—Ç–æ–π –ø—Ä–∏–º–µ—Ä –Ω–∞ Python:** –ê–ª–≥–æ—Ä–∏—Ç–º –ø–æ–∏—Å–∫–∞ –º–∞–∫—Å–∏–º–∞–ª—å–Ω–æ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞ –≤ —Å–ø–∏—Å–∫–µ.

### 5. –ü–æ–Ω—è—Ç–∏–µ —Å–ª–æ–∂–Ω–æ—Å—Ç–∏ –∞–ª–≥–æ—Ä–∏—Ç–º–æ–≤

* **–í—Ä–µ–º–µ–Ω–Ω–∞—è —Å–ª–æ–∂–Ω–æ—Å—Ç—å (Time Complexity):** –°–∫–æ–ª—å–∫–æ –≤—Ä–µ–º–µ–Ω–∏ —Ä–∞–±–æ—Ç–∞–µ—Ç –∞–ª–≥–æ—Ä–∏—Ç–º.

### 6. O-–±–æ–ª—å—à–æ–µ –∏ $\Omega$-–±–æ–ª—å—à–æ–µ

–≠—Ç–æ –∞—Å–∏–º–ø—Ç–æ—Ç–∏—á–µ—Å–∫–∏–µ –Ω–æ—Ç–∞—Ü–∏–∏ –¥–ª—è –æ—Ü–µ–Ω–∫–∏ —Å–ª–æ–∂–Ω–æ—Å—Ç–∏.

* **O-–±–æ–ª—å—à–æ–µ (Big O):** –í–µ—Ä—Ö–Ω—è—è –≥—Ä–∞–Ω–∏—Ü–∞, **—Ö—É–¥—à–∏–π —Å–ª—É—á–∞–π**. –û–ø–∏—Å—ã–≤–∞–µ—Ç, –∫–∞–∫ –º–µ–¥–ª–µ–Ω–Ω–æ –º–æ–∂–µ—Ç —Ä–∞–±–æ—Ç–∞—Ç—å –∞–ª–≥–æ—Ä–∏—Ç–º. –ù–∞–ø—Ä–∏–º–µ—Ä, $O(n^2)$ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –≤—Ä–µ–º—è —Ä–∞–±–æ—Ç—ã —Ä–∞—Å—Ç—ë—Ç –Ω–µ –±—ã—Å—Ç—Ä–µ–µ, —á–µ–º –∫–≤–∞–¥—Ä–∞—Ç –æ—Ç —Ä–∞–∑–º–µ—Ä–∞ –≤—Ö–æ–¥–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö.

* **$\Omega$-–±–æ–ª—å—à–æ–µ (Big Omega):** –ù–∏–∂–Ω—è—è –≥—Ä–∞–Ω–∏—Ü–∞, **–ª—É—á—à–∏–π —Å–ª—É—á–∞–π**. –û–ø–∏—Å—ã–≤–∞–µ—Ç, –∫–∞–∫ –±—ã—Å—Ç—Ä–æ –º–æ–∂–µ—Ç —Ä–∞–±–æ—Ç–∞—Ç—å –∞–ª–≥–æ—Ä–∏—Ç–º. –ù–∞–ø—Ä–∏–º–µ—Ä, $\Omega(n)$ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –≤ –ª—É—á—à–µ–º —Å–ª—É—á–∞–µ –∞–ª–≥–æ—Ä–∏—Ç–º –≤—ã–ø–æ–ª–Ω–∏—Ç—Å—è –Ω–µ –±—ã—Å—Ç—Ä–µ–µ, —á–µ–º –∑–∞ –ª–∏–Ω–µ–π–Ω–æ–µ –≤—Ä–µ–º—è.

### 7. –°–ª–æ–∂–Ω–æ—Å—Ç–∏ –≤–ª–æ–∂–µ–Ω–Ω—ã—Ö —Ü–∏–∫–ª–æ–≤
<a name="—Ç–≤–æ—ë_–Ω–∞–∑–≤–∞–Ω–∏–µ"></a> 
–†–∞—Å—Å–º–æ—Ç—Ä–∏–º –ø—Ä–∏–º–µ—Ä:

```python
def process_matrix(matrix):
  n = len(matrix)
  for i in range(n):      # –í–Ω–µ—à–Ω–∏–π —Ü–∏–∫–ª
    for j in range(n):    # –í–Ω—É—Ç—Ä–µ–Ω–Ω–∏–π —Ü–∏–∫–ª
      print(matrix[i][j]) # –û–ø–µ—Ä–∞—Ü–∏—è –∑–∞ O(1)
```

* **–û–ø–∏—Å–∞—Ç–µ–ª—å–Ω–∞—è —Å–ª–æ–∂–Ω–æ—Å—Ç—å:** –í—Ä–µ–º—è —Ä–∞–±–æ—Ç—ã –∞–ª–≥–æ—Ä–∏—Ç–º–∞ –ø—Ä–æ–ø–æ—Ä—Ü–∏–æ–Ω–∞–ª—å–Ω–æ –∫–≤–∞–¥—Ä–∞—Ç—É —Ä–∞–∑–º–µ—Ä–∞ –≤—Ö–æ–¥–Ω–æ–π –º–∞—Ç—Ä–∏—Ü—ã.
* **–ü—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–µ–Ω–Ω–∞—è —Å–ª–æ–∂–Ω–æ—Å—Ç—å:** –ê–ª–≥–æ—Ä–∏—Ç–º –∏—Å–ø–æ–ª—å–∑—É–µ—Ç –Ω–µ—Å–∫–æ–ª—å–∫–æ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö (`n`, `i`, `j`), –∏—Ö –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –Ω–µ –∑–∞–≤–∏—Å–∏—Ç –æ—Ç —Ä–∞–∑–º–µ—Ä–∞ `matrix`. –°–ª–æ–∂–Ω–æ—Å—Ç—å –ø–æ –ø–∞–º—è—Ç–∏ ‚Äî $O(1)$ (–∫–æ–Ω—Å—Ç–∞–Ω—Ç–Ω–∞—è).
* **–í—ã—á–∏—Å–ª–∏—Ç–µ–ª—å–Ω–∞—è (–≤—Ä–µ–º–µ–Ω–Ω–∞—è) —Å–ª–æ–∂–Ω–æ—Å—Ç—å:** –í–Ω–µ—à–Ω–∏–π —Ü–∏–∫–ª –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è `n` —Ä–∞–∑, –∏ –¥–ª—è –∫–∞–∂–¥–æ–π –∏—Ç–µ—Ä–∞—Ü–∏–∏ –≤–Ω—É—Ç—Ä–µ–Ω–Ω–∏–π —Ü–∏–∫–ª —Ç–∞–∫–∂–µ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è `n` —Ä–∞–∑. –û–±—â–µ–µ —á–∏—Å–ª–æ –æ–ø–µ—Ä–∞—Ü–∏–π: $n \times n = n^2$. –°–ª–æ–∂–Ω–æ—Å—Ç—å ‚Äî $O(n^2)$.

## –ë–∏–ª–µ—Ç 2.

### 7. –ö–≤–∞–¥—Ä–∞—Ç–∏—á–Ω—ã–µ —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∏ ($O(n^2)$) üê¢

–≠—Ç–∏ –∞–ª–≥–æ—Ä–∏—Ç–º—ã –ø—Ä–æ—Å—Ç—ã –≤ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏, –Ω–æ –Ω–µ—ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω—ã –Ω–∞ –±–æ–ª—å—à–∏—Ö –¥–∞–Ω–Ω—ã—Ö. –í –æ—Å–Ω–æ–≤–µ –ª–µ–∂–∞—Ç –≤–ª–æ–∂–µ–Ω–Ω—ã–µ —Ü–∏–∫–ª—ã.

* **–°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –ø—É–∑—ã—Ä—å–∫–æ–º (Bubble Sort):** –°–æ—Å–µ–¥–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç—ã —Å—Ä–∞–≤–Ω–∏–≤–∞—é—Ç—Å—è –∏ –º–µ–Ω—è—é—Ç—Å—è –º–µ—Å—Ç–∞–º–∏, –µ—Å–ª–∏ –æ–Ω–∏ –≤ –Ω–µ–ø—Ä–∞–≤–∏–ª—å–Ω–æ–º –ø–æ—Ä—è–¥–∫–µ.
* **–°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –≤—ã–±–æ—Ä–æ–º (Selection Sort):** –ù–∞—Ö–æ–¥–∏–º –º–∏–Ω–∏–º–∞–ª—å–Ω—ã–π —ç–ª–µ–º–µ–Ω—Ç –∏ —Å—Ç–∞–≤–∏–º –µ–≥–æ –≤ –Ω–∞—á–∞–ª–æ, –∑–∞—Ç–µ–º –∏—â–µ–º —Å–ª–µ–¥—É—é—â–∏–π –º–∏–Ω–∏–º–∞–ª—å–Ω—ã–π –¥–ª—è –æ—Å—Ç–∞–≤—à–µ–π—Å—è —á–∞—Å—Ç–∏ –∏ —Ç.–¥.
* **–°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –≤—Å—Ç–∞–≤–∫–∞–º–∏ (Insertion Sort):** –ë–µ—Ä—ë–º —ç–ª–µ–º–µ–Ω—Ç—ã –ø–æ –æ–¥–Ω–æ–º—É –∏ "–≤—Å—Ç–∞–≤–ª—è–µ–º" –∏—Ö –Ω–∞ –ø—Ä–∞–≤–∏–ª—å–Ω–æ–µ –º–µ—Å—Ç–æ –≤ —É–∂–µ –æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–π —á–∞—Å—Ç–∏ –º–∞—Å—Å–∏–≤–∞.

**–ü—Ä–∏–º–µ—Ä: –°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –ø—É–∑—ã—Ä—å–∫–æ–º**
```python
def bubble_sort(arr):
  n = len(arr)
  for i in range(n):
    for j in range(0, n-i-1):
      if arr[j] > arr[j+1]:
        arr[j], arr[j+1] = arr[j+1], arr[j] # –û–±–º–µ–Ω
  return arr

print(bubble_sort([64, 34, 25, 12, 22, 11, 90]))
```

### 8. –ë—ã—Å—Ç—Ä—ã–µ —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∏ ($O(n \log n)$) üöÄ

–ì–æ—Ä–∞–∑–¥–æ —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–µ–µ. –ò—Å–ø–æ–ª—å–∑—É—é—Ç –ø—Ä–∏–Ω—Ü–∏–ø **"—Ä–∞–∑–¥–µ–ª—è–π –∏ –≤–ª–∞—Å—Ç–≤—É–π"**.

* **–°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ —Å–ª–∏—è–Ω–∏–µ–º (Merge Sort):** –ú–∞—Å—Å–∏–≤ —Ä–µ–∫—É—Ä—Å–∏–≤–Ω–æ –¥–µ–ª–∏—Ç—Å—è –ø–æ–ø–æ–ª–∞–º –¥–æ —Ç–µ—Ö –ø–æ—Ä, –ø–æ–∫–∞ –Ω–µ –æ—Å—Ç–∞–Ω—É—Ç—Å—è –µ–¥–∏–Ω–∏—á–Ω—ã–µ —ç–ª–µ–º–µ–Ω—Ç—ã. –ó–∞—Ç–µ–º –æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ —á–∞—Å—Ç–∏ "—Å–ª–∏–≤–∞—é—Ç—Å—è" –≤–º–µ—Å—Ç–µ.
* **–ë—ã—Å—Ç—Ä–∞—è —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ (Quick Sort):** –í—ã–±–∏—Ä–∞–µ—Ç—Å—è "–æ–ø–æ—Ä–Ω—ã–π" —ç–ª–µ–º–µ–Ω—Ç (pivot). –ú–∞—Å—Å–∏–≤ –¥–µ–ª–∏—Ç—Å—è –Ω–∞ –¥–≤–µ —á–∞—Å—Ç–∏: —ç–ª–µ–º–µ–Ω—Ç—ã –º–µ–Ω—å—à–µ –æ–ø–æ—Ä–Ω–æ–≥–æ –∏ —ç–ª–µ–º–µ–Ω—Ç—ã –±–æ–ª—å—à–µ –æ–ø–æ—Ä–Ω–æ–≥–æ. –ó–∞—Ç–µ–º —á–∞—Å—Ç–∏ —Ä–µ–∫—É—Ä—Å–∏–≤–Ω–æ —Å–æ—Ä—Ç–∏—Ä—É—é—Ç—Å—è.

**–ü—Ä–∏–º–µ—Ä: –ë—ã—Å—Ç—Ä–∞—è —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞**
```python
def quick_sort(arr):
  if len(arr) <= 1:
    return arr
  else:
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)

print(quick_sort([64, 34, 25, 12, 22, 11, 90]))
```

## –ë–∏–ª–µ—Ç 3. 
–†–µ–∫—É—Ä—Å–∏–≤–Ω—ã–µ –º–µ—Ç–æ–¥—ã –≤ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏–∏. –û—Å–Ω–æ–≤–Ω—ã–µ –ø–æ–Ω—è—Ç–∏—è, —Ä–µ–∫—É—Ä—Å–∏–≤–Ω—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏ –∏ –ø—Ä–æ—Ü–µ–¥—É—Ä—ã. –î–µ–∫–æ–º–ø–æ–∑–∏—Ü–∏–æ–Ω–Ω—ã–π –ø–æ–¥—Ö–æ–¥ –∫ —Å–∏–Ω—Ç–µ–∑—É –∞–ª–≥–æ—Ä–∏—Ç–º–æ–≤ –Ω–∞ –ø—Ä–∏–º–µ—Ä–µ –∑–∞–¥–∞—á–∏ –æ –•–∞–Ω–æ–π—Å–∫–∏—Ö –±–∞—à–Ω—è—Ö.

### 1. –û—Å–Ω–æ–≤–Ω—ã–µ –ø–æ–Ω—è—Ç–∏—è —Ä–µ–∫—É—Ä—Å–∏–∏

**–†–µ–∫—É—Ä—Å–∏—è** ‚Äî —ç—Ç–æ –º–µ—Ç–æ–¥ –≤ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏–∏, –ø—Ä–∏ –∫–æ—Ç–æ—Ä–æ–º —Ñ—É–Ω–∫—Ü–∏—è –≤—ã–∑—ã–≤–∞–µ—Ç —Å–∞–º—É —Å–µ–±—è –¥–ª—è —Ä–µ—à–µ–Ω–∏—è –∑–∞–¥–∞—á–∏.

–ö–ª—é—á–µ–≤—ã–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã –ª—é–±–æ–π —Ä–µ–∫—É—Ä—Å–∏–≤–Ω–æ–π —Ñ—É–Ω–∫—Ü–∏–∏:
* **–ë–∞–∑–æ–≤—ã–π —Å–ª—É—á–∞–π (—É—Å–ª–æ–≤–∏–µ –≤—ã—Ö–æ–¥–∞):** –ü—Ä–æ—Å—Ç–æ–µ —É—Å–ª–æ–≤–∏–µ, –ø—Ä–∏ –∫–æ—Ç–æ—Ä–æ–º —Ñ—É–Ω–∫—Ü–∏—è –ø—Ä–µ–∫—Ä–∞—â–∞–µ—Ç –≤—ã–∑—ã–≤–∞—Ç—å —Å–µ–±—è –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –∫–æ–Ω–∫—Ä–µ—Ç–Ω—ã–π —Ä–µ–∑—É–ª—å—Ç–∞—Ç. –≠—Ç–æ –ø—Ä–µ–¥–æ—Ç–≤—Ä–∞—â–∞–µ—Ç –±–µ—Å–∫–æ–Ω–µ—á–Ω—ã–π —Ü–∏–∫–ª.

–ê–Ω–∞–ª–æ–≥–∏—è –∏–∑ –∂–∏–∑–Ω–∏ ‚Äî **–º–∞—Ç—Ä—ë—à–∫–∞**. –ß—Ç–æ–±—ã –æ—Ç–∫—Ä—ã—Ç—å –≤—Å—é –º–∞—Ç—Ä—ë—à–∫—É (–∑–∞–¥–∞—á–∞), –Ω—É–∂–Ω–æ –æ—Ç–∫—Ä—ã—Ç—å —Ç–µ–∫—É—â—É—é –∫—É–∫–ª—É (—Ä–µ–∫—É—Ä—Å–∏–≤–Ω—ã–π —à–∞–≥) –∏ –æ–±–Ω–∞—Ä—É–∂–∏—Ç—å –≤–Ω—É—Ç—Ä–∏ –Ω–µ—ë –º–∞—Ç—Ä—ë—à–∫—É –ø–æ–º–µ–Ω—å—à–µ (–ø–æ–¥–∑–∞–¥–∞—á–∞). –¢–∞–∫ –ø—Ä–æ–¥–æ–ª–∂–∞–µ—Ç—Å—è –¥–æ —Ç–µ—Ö –ø–æ—Ä, –ø–æ–∫–∞ –º—ã –Ω–µ –¥–æ–π–¥–µ–º –¥–æ —Å–∞–º–æ–π –º–∞–ª–µ–Ω—å–∫–æ–π, –Ω–µ—Ä–∞–∑–±–æ—Ä–Ω–æ–π –∫—É–∫–ª—ã (–±–∞–∑–æ–≤—ã–π —Å–ª—É—á–∞–π).

### 2. –†–µ–∫—É—Ä—Å–∏–≤–Ω—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏ –∏ –ø—Ä–æ—Ü–µ–¥—É—Ä—ã

* **–†–µ–∫—É—Ä—Å–∏–≤–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è** ‚Äî —ç—Ç–æ —Ñ—É–Ω–∫—Ü–∏—è, –∫–æ—Ç–æ—Ä–∞—è –≤—ã–∑—ã–≤–∞–µ—Ç —Å–∞–º—É —Å–µ–±—è –∏ **–≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –∑–Ω–∞—á–µ–Ω–∏–µ**.
* **–†–µ–∫—É—Ä—Å–∏–≤–Ω–∞—è –ø—Ä–æ—Ü–µ–¥—É—Ä–∞** ‚Äî —ç—Ç–æ –ø—Ä–æ—Ü–µ–¥—É—Ä–∞ (–≤ Python —Ç–æ–∂–µ `def`), –∫–æ—Ç–æ—Ä–∞—è –≤—ã–∑—ã–≤–∞–µ—Ç —Å–∞–º—É —Å–µ–±—è, –Ω–æ **–Ω–µ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –∑–Ω–∞—á–µ–Ω–∏–µ**, –∞ –≤—ã–ø–æ–ª–Ω—è–µ—Ç –∫–∞–∫–æ–µ-–ª–∏–±–æ –¥–µ–π—Å—Ç–≤–∏–µ (–Ω–∞–ø—Ä–∏–º–µ—Ä, –ø–µ—á–∞—Ç—å –Ω–∞ —ç–∫—Ä–∞–Ω).

**–ö–ª–∞—Å—Å–∏—á–µ—Å–∫–∏–π –ø—Ä–∏–º–µ—Ä —Ä–µ–∫—É—Ä—Å–∏–≤–Ω–æ–π —Ñ—É–Ω–∫—Ü–∏–∏ ‚Äî –≤—ã—á–∏—Å–ª–µ–Ω–∏–µ —Ñ–∞–∫—Ç–æ—Ä–∏–∞–ª–∞ $n!$.**

* **–ó–∞–¥–∞—á–∞:** –í—ã—á–∏—Å–ª–∏—Ç—å $n! = n \cdot (n-1) \cdot \dots \cdot 1$.
* **–ë–∞–∑–æ–≤—ã–π —Å–ª—É—á–∞–π:** –ï—Å–ª–∏ $n=0$ –∏–ª–∏ $n=1$, —Ç–æ —Ñ–∞–∫—Ç–æ—Ä–∏–∞–ª —Ä–∞–≤–µ–Ω 1.
* **–†–µ–∫—É—Ä—Å–∏–≤–Ω—ã–π —à–∞–≥:** –î–ª—è –ª—é–±–æ–≥–æ –¥—Ä—É–≥–æ–≥–æ $n$, —Ñ–∞–∫—Ç–æ—Ä–∏–∞–ª —Ä–∞–≤–µ–Ω $n \cdot (n-1)!$.

```python
def factorial(n):
  # –ë–∞–∑–æ–≤—ã–π —Å–ª—É—á–∞–π
  if n <= 1:
    return 1
  # –†–µ–∫—É—Ä—Å–∏–≤–Ω—ã–π —à–∞–≥
  else:
    return n * factorial(n - 1)

print(f"5! = {factorial(5)}") # -> 5! = 120
```

### 3. –î–µ–∫–æ–º–ø–æ–∑–∏—Ü–∏—è –Ω–∞ –ø—Ä–∏–º–µ—Ä–µ –•–∞–Ω–æ–π—Å–∫–∏—Ö –±–∞—à–µ–Ω

**–î–µ–∫–æ–º–ø–æ–∑–∏—Ü–∏–æ–Ω–Ω—ã–π –ø–æ–¥—Ö–æ–¥** ‚Äî —ç—Ç–æ –º–µ—Ç–æ–¥ —Ä–µ—à–µ–Ω–∏—è —Å–ª–æ–∂–Ω–æ–π –∑–∞–¥–∞—á–∏ –ø—É—Ç—ë–º –µ—ë —Ä–∞–∑–±–∏–µ–Ω–∏—è –Ω–∞ –±–æ–ª–µ–µ –º–µ–ª–∫–∏–µ –∏ –ø—Ä–æ—Å—Ç—ã–µ **–ø–æ–¥–∑–∞–¥–∞—á–∏ —Ç–æ–≥–æ –∂–µ —Ç–∏–ø–∞**. –†–µ–∫—É—Ä—Å–∏—è –∏–¥–µ–∞–ª—å–Ω–æ –ø–æ–¥—Ö–æ–¥–∏—Ç –¥–ª—è —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ —ç—Ç–æ–≥–æ –ø–æ–¥—Ö–æ–¥–∞.

**–ó–∞–¥–∞—á–∞ –æ –•–∞–Ω–æ–π—Å–∫–∏—Ö –±–∞—à–Ω—è—Ö:**
–ü–µ—Ä–µ–º–µ—Å—Ç–∏—Ç—å —Å—Ç–æ–ø–∫—É –∏–∑ `n` –¥–∏—Å–∫–æ–≤ —Å –æ–¥–Ω–æ–≥–æ —Å—Ç–µ—Ä–∂–Ω—è (`A`) –Ω–∞ –¥—Ä—É–≥–æ–π (`C`), –∏—Å–ø–æ–ª—å–∑—É—è —Ç—Ä–µ—Ç–∏–π –≤—Å–ø–æ–º–æ–≥–∞—Ç–µ–ª—å–Ω—ã–π —Å—Ç–µ—Ä–∂–µ–Ω—å (`B`).

**–ü—Ä–∞–≤–∏–ª–∞:**
1.  –ó–∞ —Ä–∞–∑ –º–æ–∂–Ω–æ –ø–µ—Ä–µ–º–µ—â–∞—Ç—å —Ç–æ–ª—å–∫–æ –æ–¥–∏–Ω –¥–∏—Å–∫.
2.  –ù–µ–ª—å–∑—è –∫–ª–∞—Å—Ç—å –±–æ–ª—å—à–∏–π –¥–∏—Å–∫ –Ω–∞ –º–µ–Ω—å—à–∏–π.

**–î–µ–∫–æ–º–ø–æ–∑–∏—Ü–∏—è –∏ —Ä–µ–∫—É—Ä—Å–∏–≤–Ω—ã–π —Å–∏–Ω—Ç–µ–∑:**
–ß—Ç–æ–±—ã –ø–µ—Ä–µ–º–µ—Å—Ç–∏—Ç—å `n` –¥–∏—Å–∫–æ–≤ —Å–æ —Å—Ç–µ—Ä–∂–Ω—è `A` –Ω–∞ `C`, –Ω—É–∂–Ω–æ:

1.  **–†–µ—à–∏—Ç—å –ø–æ–¥–∑–∞–¥–∞—á—É:** –ü–µ—Ä–µ–º–µ—Å—Ç–∏—Ç—å —Å—Ç–æ–ø–∫—É –ø–æ–º–µ–Ω—å—à–µ –∏–∑ `n-1` –¥–∏—Å–∫–æ–≤ —Å–æ —Å—Ç–µ—Ä–∂–Ω—è `A` –Ω–∞ –≤—Å–ø–æ–º–æ–≥–∞—Ç–µ–ª—å–Ω—ã–π `B`.
2.  **–í—ã–ø–æ–ª–Ω–∏—Ç—å –±–∞–∑–æ–≤–æ–µ –¥–µ–π—Å—Ç–≤–∏–µ:** –ü–µ—Ä–µ–º–µ—Å—Ç–∏—Ç—å —Å–∞–º—ã–π –±–æ–ª—å—à–æ–π, `n`-—ã–π –¥–∏—Å–∫ —Å–æ —Å—Ç–µ—Ä–∂–Ω—è `A` –Ω–∞ —Ü–µ–ª–µ–≤–æ–π `C`.
3.  **–†–µ—à–∏—Ç—å –ø–æ–¥–∑–∞–¥–∞—á—É:** –ü–µ—Ä–µ–º–µ—Å—Ç–∏—Ç—å —Å—Ç–æ–ø–∫—É –∏–∑ `n-1` –¥–∏—Å–∫–æ–≤ —Å–æ –≤—Å–ø–æ–º–æ–≥–∞—Ç–µ–ª—å–Ω–æ–≥–æ —Å—Ç–µ—Ä–∂–Ω—è `B` –Ω–∞ —Ü–µ–ª–µ–≤–æ–π `C`.

–≠—Ç–æ—Ç –∞–ª–≥–æ—Ä–∏—Ç–º —ç–ª–µ–≥–∞–Ω—Ç–Ω–æ —Ä–µ–∞–ª–∏–∑—É–µ—Ç—Å—è —á–µ—Ä–µ–∑ —Ä–µ–∫—É—Ä—Å–∏–≤–Ω—É—é –ø—Ä–æ—Ü–µ–¥—É—Ä—É.

  """
  –†–µ–∫—É—Ä—Å–∏–≤–Ω–∞—è –ø—Ä–æ—Ü–µ–¥—É—Ä–∞ –¥–ª—è —Ä–µ—à–µ–Ω–∏—è –∑–∞–¥–∞—á–∏ –æ –•–∞–Ω–æ–π—Å–∫–∏—Ö –±–∞—à–Ω—è—Ö.

  if n > 0:
    # 1. –ü–µ—Ä–µ–º–µ—â–∞–µ–º n-1 –¥–∏—Å–∫–æ–≤ —Å –ò–°–¢–û–ß–ù–ò–ö–ê –Ω–∞ –í–°–ü–û–ú–û–ì–ê–¢–ï–õ–¨–ù–´–ô
    hanoi(n - 1, source, auxiliary, destination)
    
    # 2. –ü–µ—Ä–µ–º–µ—â–∞–µ–º –æ—Å—Ç–∞–≤—à–∏–π—Å—è —Å–∞–º—ã–π –±–æ–ª—å—à–æ–π –¥–∏—Å–∫ —Å –ò–°–¢–û–ß–ù–ò–ö–ê –Ω–∞ –¶–ï–õ–ï–í–û–ô
    print(f"–ü–µ—Ä–µ–º–µ—Å—Ç–∏—Ç—å –¥–∏—Å–∫ {n} —Å {source} –Ω–∞ {destination}")
    
    # 3. –ü–µ—Ä–µ–º–µ—â–∞–µ–º n-1 –¥–∏—Å–∫–æ–≤ —Å–æ –í–°–ü–û–ú–û–ì–ê–¢–ï–õ–¨–ù–û–ì–û –Ω–∞ –¶–ï–õ–ï–í–û–ô
    hanoi(n - 1, auxiliary, destination, source)

# –ü—Ä–∏–º–µ—Ä –≤—ã–∑–æ–≤–∞ –¥–ª—è 3 –¥–∏—Å–∫–æ–≤
print("–ü–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —Ö–æ–¥–æ–≤ –¥–ª—è 3 –¥–∏—Å–∫–æ–≤:")

## –ë–∏–ª–µ—Ç 4. 
–ß–∏—Å–ª–µ–Ω–Ω–æ–µ —Ä–µ—à–µ–Ω–∏–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–π. –ë–∏–Ω–∞—Ä–Ω—ã–π –ø–æ–∏—Å–∫. –≠–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω–æ—Å—Ç—å —Ä–µ–∫—É—Ä—Å–∏–∏ –∏ –∏—Ç–µ—Ä–∞—Ü–∏–∏ –Ω–∞ –ø—Ä–∏–º–µ—Ä–µ –≤—ã—á–∏—Å–ª–µ–Ω–∏—è —Ñ–∞–∫—Ç–æ—Ä–∏–∞–ª–∞.

–ö–æ–Ω–µ—á–Ω–æ, –≤–æ—Ç —Ä–∞–∑–±–æ—Ä —á–µ—Ç–≤—ë—Ä—Ç–æ–≥–æ –±–∏–ª–µ—Ç–∞.

### 1. –ß–∏—Å–ª–µ–Ω–Ω–æ–µ —Ä–µ—à–µ–Ω–∏–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–π: –ú–µ—Ç–æ–¥ –¥–µ–ª–µ–Ω–∏—è –ø–æ–ø–æ–ª–∞–º

**–ú–µ—Ç–æ–¥ –¥–µ–ª–µ–Ω–∏—è –ø–æ–ø–æ–ª–∞–º (–±–∏—Å–µ–∫—Ü–∏–∏)** ‚Äî —ç—Ç–æ —á–∏—Å–ª–µ–Ω–Ω—ã–π –∞–ª–≥–æ—Ä–∏—Ç–º –¥–ª—è –Ω–∞—Ö–æ–∂–¥–µ–Ω–∏—è –∫–æ—Ä–Ω—è —É—Ä–∞–≤–Ω–µ–Ω–∏—è $f(x) = 0$. –û–Ω —Ä–∞–±–æ—Ç–∞–µ—Ç –Ω–∞ **–Ω–µ–ø—Ä–µ—Ä—ã–≤–Ω–æ–π —Ñ—É–Ω–∫—Ü–∏–∏** –∏ –ø–æ —Å–≤–æ–µ–π —Å—É—Ç–∏ —è–≤–ª—è–µ—Ç—Å—è **–±–∏–Ω–∞—Ä–Ω—ã–º –ø–æ–∏—Å–∫–æ–º** –¥–ª—è –ø–æ–∏—Å–∫–∞ –∫–æ—Ä–Ω—è.

**–ò–¥–µ—è:**
1.  –ë–µ—Ä—ë—Ç—Å—è –æ—Ç—Ä–µ–∑–æ–∫ $[a, b]$, –Ω–∞ –∫–æ–Ω—Ü–∞—Ö –∫–æ—Ç–æ—Ä–æ–≥–æ —Ñ—É–Ω–∫—Ü–∏—è $f(x)$ –∏–º–µ–µ—Ç **—Ä–∞–∑–Ω—ã–µ –∑–Ω–∞–∫–∏** (—Ç.–µ., $f(a) \cdot f(b) < 0$). –≠—Ç–æ –≥–∞—Ä–∞–Ω—Ç–∏—Ä—É–µ—Ç, —á—Ç–æ –≤–Ω—É—Ç—Ä–∏ –æ—Ç—Ä–µ–∑–∫–∞ –µ—Å—Ç—å —Ö–æ—Ç—è –±—ã –æ–¥–∏–Ω –∫–æ—Ä–µ–Ω—å.
2.  –ù–∞—Ö–æ–¥–∏—Ç—Å—è —Å–µ—Ä–µ–¥–∏–Ω–∞ –æ—Ç—Ä–µ–∑–∫–∞ $c = (a + b) / 2$.
3.  –í—ã—á–∏—Å–ª—è–µ—Ç—Å—è –∑–Ω–∞—á–µ–Ω–∏–µ $f(c)$.
4.  –ï—Å–ª–∏ $f(c)$ –∏–º–µ–µ—Ç —Ç–æ—Ç –∂–µ –∑–Ω–∞–∫, —á—Ç–æ –∏ $f(a)$, —Ç–æ –∫–æ—Ä–µ–Ω—å –Ω–∞—Ö–æ–¥–∏—Ç—Å—è –Ω–∞ –æ—Ç—Ä–µ–∑–∫–µ $[c, b]$. –í –ø—Ä–æ—Ç–∏–≤–Ω–æ–º —Å–ª—É—á–∞–µ ‚Äî –Ω–∞ –æ—Ç—Ä–µ–∑–∫–µ $[a, c]$.
5.  –ü—Ä–æ—Ü–µ—Å—Å –ø–æ–≤—Ç–æ—Ä—è–µ—Ç—Å—è, —Å—É–∂–∞—è –æ—Ç—Ä–µ–∑–æ–∫ –≤–¥–≤–æ–µ –Ω–∞ –∫–∞–∂–¥–æ–π –∏—Ç–µ—Ä–∞—Ü–∏–∏, –ø–æ–∫–∞ –Ω–µ –±—É–¥–µ—Ç –¥–æ—Å—Ç–∏–≥–Ω—É—Ç–∞ –Ω—É–∂–Ω–∞—è —Ç–æ—á–Ω–æ—Å—Ç—å.

**–°–ª–æ–∂–Ω–æ—Å—Ç—å:** –ö–æ–ª–∏—á–µ—Å—Ç–≤–æ –∏—Ç–µ—Ä–∞—Ü–∏–π –¥–ª—è –¥–æ—Å—Ç–∏–∂–µ–Ω–∏—è —Ç–æ—á–Ω–æ—Å—Ç–∏ $\epsilon$ —Ä–∞–≤–Ω–æ $O(\log_2((b-a)/\epsilon))$.

```python
def bisection_method(f, a, b, tolerance=1e-7):
  """–ù–∞—Ö–æ–¥–∏—Ç –∫–æ—Ä–µ–Ω—å —Ñ—É–Ω–∫—Ü–∏–∏ f –Ω–∞ –æ—Ç—Ä–µ–∑–∫–µ [a, b] –º–µ—Ç–æ–¥–æ–º –¥–µ–ª–µ–Ω–∏—è –ø–æ–ø–æ–ª–∞–º."""
  if f(a) * f(b) >= 0:
    return "–ù–∞ –æ—Ç—Ä–µ–∑–∫–µ –Ω–µ—Ç –∫–æ—Ä–Ω—è –∏–ª–∏ –∏—Ö —á–µ—Ç–Ω–æ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ."
  
  while (b - a) / 2.0 > tolerance:
    midpoint = (a + b) / 2.0
    if f(midpoint) == 0:
      return midpoint # –ö–æ—Ä–µ–Ω—å –Ω–∞–π–¥–µ–Ω —Ç–æ—á–Ω–æ
    elif f(a) * f(midpoint) < 0:
      b = midpoint # –ö–æ—Ä–µ–Ω—å –≤ –ª–µ–≤–æ–π –ø–æ–ª–æ–≤–∏–Ω–µ
    else:
      a = midpoint # –ö–æ—Ä–µ–Ω—å –≤ –ø—Ä–∞–≤–æ–π –ø–æ–ª–æ–≤–∏–Ω–µ
      
  return (a + b) / 2.0

# –ù–∞–π–¥–µ–º –∫–æ—Ä–µ–Ω—å —É—Ä–∞–≤–Ω–µ–Ω–∏—è x^2 - 4 = 0 –Ω–∞ –æ—Ç—Ä–µ–∑–∫–µ [0, 5]
def my_function(x):
  return x**2 - 4

root = bisection_method(my_function, 0, 5)
print(f"–ö–æ—Ä–µ–Ω—å —É—Ä–∞–≤–Ω–µ–Ω–∏—è x^2 - 4 = 0 –ø—Ä–∏–º–µ—Ä–Ω–æ —Ä–∞–≤–µ–Ω: {root:.7f}")
# -> –ö–æ—Ä–µ–Ω—å —É—Ä–∞–≤–Ω–µ–Ω–∏—è x^2 - 4 = 0 –ø—Ä–∏–º–µ—Ä–Ω–æ —Ä–∞–≤–µ–Ω: 2.0000000
```

### 2. –ë–∏–Ω–∞—Ä–Ω—ã–π –ø–æ–∏—Å–∫

**–ë–∏–Ω–∞—Ä–Ω—ã–π –ø–æ–∏—Å–∫** ‚Äî —ç—Ç–æ –≤—ã—Å–æ–∫–æ—ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω—ã–π –∞–ª–≥–æ—Ä–∏—Ç–º –ø–æ–∏—Å–∫–∞ —ç–ª–µ–º–µ–Ω—Ç–∞ –≤ **–æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–º –º–∞—Å—Å–∏–≤–µ**. üïµÔ∏è‚Äç‚ôÇÔ∏è

**–ê–ª–≥–æ—Ä–∏—Ç–º:**
1.  –û–ø—Ä–µ–¥–µ–ª—è–µ–º –ª–µ–≤—É—é (`left`) –∏ –ø—Ä–∞–≤—É—é (`right`) –≥—Ä–∞–Ω–∏—Ü—ã –ø–æ–∏—Å–∫–∞.
2.  –ù–∞—Ö–æ–¥–∏–º —Å—Ä–µ–¥–Ω–∏–π —ç–ª–µ–º–µ–Ω—Ç `mid = (left + right) // 2`.
3.  –°—Ä–∞–≤–Ω–∏–≤–∞–µ–º –∏—Å–∫–æ–º—ã–π —ç–ª–µ–º–µ–Ω—Ç —Å `arr[mid]`:
    * –ï—Å–ª–∏ –æ–Ω–∏ —Ä–∞–≤–Ω—ã ‚Äî —ç–ª–µ–º–µ–Ω—Ç –Ω–∞–π–¥–µ–Ω.
    * –ï—Å–ª–∏ –∏—Å–∫–æ–º—ã–π —ç–ª–µ–º–µ–Ω—Ç **–º–µ–Ω—å—à–µ** ‚Äî —Å–¥–≤–∏–≥–∞–µ–º –ø—Ä–∞–≤—É—é –≥—Ä–∞–Ω–∏—Ü—É `right = mid - 1` –∏ –∏—â–µ–º –≤ –ª–µ–≤–æ–π –ø–æ–ª–æ–≤–∏–Ω–µ.
    * –ï—Å–ª–∏ –∏—Å–∫–æ–º—ã–π —ç–ª–µ–º–µ–Ω—Ç **–±–æ–ª—å—à–µ** ‚Äî —Å–¥–≤–∏–≥–∞–µ–º –ª–µ–≤—É—é –≥—Ä–∞–Ω–∏—Ü—É `left = mid + 1` –∏ –∏—â–µ–º –≤ –ø—Ä–∞–≤–æ–π –ø–æ–ª–æ–≤–∏–Ω–µ.
4.  –®–∞–≥–∏ 2-3 –ø–æ–≤—Ç–æ—Ä—è—é—Ç—Å—è, –ø–æ–∫–∞ `left <= right`. –ï—Å–ª–∏ —Ü–∏–∫–ª –∑–∞–≤–µ—Ä—à–∏–ª—Å—è, –∞ —ç–ª–µ–º–µ–Ω—Ç –Ω–µ –Ω–∞–π–¥–µ–Ω, –∑–Ω–∞—á–∏—Ç, –µ–≥–æ –Ω–µ—Ç –≤ –º–∞—Å—Å–∏–≤–µ.

**–°–ª–æ–∂–Ω–æ—Å—Ç—å:** –í—Ä–µ–º–µ–Ω–Ω–∞—è —Å–ª–æ–∂–Ω–æ—Å—Ç—å ‚Äî $O(\log n)$, –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–µ–Ω–Ω–∞—è ‚Äî $O(1)$.

```python
def binary_search(arr, target):
  """–ò—â–µ—Ç target –≤ –æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–º –º–∞—Å—Å–∏–≤–µ arr –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –µ–≥–æ –∏–Ω–¥–µ–∫—Å."""
  left, right = 0, len(arr) - 1
  
  while left <= right:
    mid = (left + right) // 2
    if arr[mid] == target:
      return mid # –≠–ª–µ–º–µ–Ω—Ç –Ω–∞–π–¥–µ–Ω
    elif arr[mid] < target:
      left = mid + 1 # –ò—Å–∫–∞—Ç—å –≤ –ø—Ä–∞–≤–æ–π –ø–æ–ª–æ–≤–∏–Ω–µ
    else:
      right = mid - 1 # –ò—Å–∫–∞—Ç—å –≤ –ª–µ–≤–æ–π –ø–æ–ª–æ–≤–∏–Ω–µ
      
  return -1 # –≠–ª–µ–º–µ–Ω—Ç –Ω–µ –Ω–∞–π–¥–µ–Ω

sorted_array = [2, 5, 8, 12, 16, 23, 38, 56, 72, 91]
index = binary_search(sorted_array, 23)
print(f"–≠–ª–µ–º–µ–Ω—Ç 23 –Ω–∞—Ö–æ–¥–∏—Ç—Å—è –ø–æ –∏–Ω–¥–µ–∫—Å—É: {index}") # -> 5
```

### 3. –≠–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω–æ—Å—Ç—å —Ä–µ–∫—É—Ä—Å–∏–∏ –∏ –∏—Ç–µ—Ä–∞—Ü–∏–∏

–õ—é–±–æ–π –∞–ª–≥–æ—Ä–∏—Ç–º, —Ä–µ–∞–ª–∏–∑–æ–≤–∞–Ω–Ω—ã–π —Å –ø–æ–º–æ—â—å—é **—Ä–µ–∫—É—Ä—Å–∏–∏**, –º–æ–∂–Ω–æ —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å —Å –ø–æ–º–æ—â—å—é **–∏—Ç–µ—Ä–∞—Ü–∏–∏ (—Ü–∏–∫–ª–æ–≤)**, –∏ –Ω–∞–æ–±–æ—Ä–æ—Ç. –í—ã–±–æ—Ä –º–µ–∂–¥—É –Ω–∏–º–∏ ‚Äî —ç—Ç–æ –≤–æ–ø—Ä–æ—Å —É–¥–æ–±—Å—Ç–≤–∞, —á–∏—Ç–∞–µ–º–æ—Å—Ç–∏ –∫–æ–¥–∞ –∏ —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ—Å—Ç–∏.

* **–†–µ–∫—É—Ä—Å–∏—è:** –ö–æ–¥ —á–∞—Å—Ç–æ –±–æ–ª–µ–µ –ª–∞–∫–æ–Ω–∏—á–Ω—ã–π –∏ –±–ª–∏–∂–µ –∫ –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–æ–º—É –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—é. –û–¥–Ω–∞–∫–æ –º–æ–∂–µ—Ç –ø—Ä–∏–≤–æ–¥–∏—Ç—å –∫ –æ—à–∏–±–∫–µ –ø–µ—Ä–µ–ø–æ–ª–Ω–µ–Ω–∏—è —Å—Ç–µ–∫–∞ (`Stack Overflow`) –∏ –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã–º —Ä–∞—Å—Ö–æ–¥–∞–º –Ω–∞ –≤—ã–∑–æ–≤—ã —Ñ—É–Ω–∫—Ü–∏–π.
* **–ò—Ç–µ—Ä–∞—Ü–∏—è:** –û–±—ã—á–Ω–æ —Ä–∞–±–æ—Ç–∞–µ—Ç –±—ã—Å—Ç—Ä–µ–µ –∏ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç –º–µ–Ω—å—à–µ –ø–∞–º—è—Ç–∏ (–Ω–µ —Ä–∞—Å—Ö–æ–¥—É–µ—Ç —Å—Ç–µ–∫ –≤—ã–∑–æ–≤–æ–≤). –ö–æ–¥ –º–æ–∂–µ—Ç –±—ã—Ç—å –±–æ–ª–µ–µ –≥—Ä–æ–º–æ–∑–¥–∫–∏–º.

**–ü—Ä–∏–º–µ—Ä: –≤—ã—á–∏—Å–ª–µ–Ω–∏–µ —Ñ–∞–∫—Ç–æ—Ä–∏–∞–ª–∞**

**–†–µ–∫—É—Ä—Å–∏–≤–Ω–∞—è —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è (–∏–∑ –±–∏–ª–µ—Ç–∞ 3):**

def factorial_recursive(n):
  if n <= 1:
    return 1
  else:
    return n * factorial_recursive(n - 1)

  result = 1
  for i in range(1, n + 1):
    result *= i
  return result

### 2. –ú–µ—Ç–æ–¥ —É—Å—Ä–µ–¥–Ω–µ–Ω–∏—è (Aggregate Method)

–≠—Ç–æ —Å–∞–º—ã–π –ø—Ä–æ—Å—Ç–æ–π –º–µ—Ç–æ–¥. –ú—ã –Ω–∞—Ö–æ–¥–∏–º **–æ–±—â—É—é —Å—Ç–æ–∏–º–æ—Å—Ç—å** $T(n)$ –¥–ª—è –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ –∏–∑ $n$ –æ–ø–µ—Ä–∞—Ü–∏–π –≤ —Ö—É–¥—à–µ–º —Å–ª—É—á–∞–µ. –ó–∞—Ç–µ–º –∞–º–æ—Ä—Ç–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω–∞—è —Å—Ç–æ–∏–º–æ—Å—Ç—å –æ–¥–Ω–æ–π –æ–ø–µ—Ä–∞—Ü–∏–∏ –≤—ã—á–∏—Å–ª—è–µ—Ç—Å—è –∫–∞–∫ $T(n) / n$.

**–ê–Ω–∞–ª–∏–∑:** –ó–∞ $n$ –æ–ø–µ—Ä–∞—Ü–∏–π –¥–æ–±–∞–≤–ª–µ–Ω–∏—è –æ–±—â–µ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∫–æ–ø–∏—Ä–æ–≤–∞–Ω–∏–π –Ω–µ –ø—Ä–µ–≤—ã—Å–∏—Ç $n$. –¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, –æ–±—â–∞—è —Å—Ç–æ–∏–º–æ—Å—Ç—å –≤—Å–µ—Ö –æ–ø–µ—Ä–∞—Ü–∏–π (–æ–±—ã—á–Ω—ã—Ö –¥–æ–±–∞–≤–ª–µ–Ω–∏–π –∏ –∫–æ–ø–∏—Ä–æ–≤–∞–Ω–∏–π) –±—É–¥–µ—Ç $O(n)$.
* **–û–±—â–∞—è —Å—Ç–æ–∏–º–æ—Å—Ç—å $T(n)$:** $O(n)$

### 4. –ú–µ—Ç–æ–¥ –ø–æ—Ç–µ–Ω—Ü–∏–∞–ª–æ–≤ (Potential Method)

–≠—Ç–æ —Å–∞–º—ã–π –≥–∏–±–∫–∏–π –º–µ—Ç–æ–¥, –ø–æ—Ö–æ–∂–∏–π –Ω–∞ –º–µ—Ç–æ–¥ –ø—Ä–µ–¥–æ–ø–ª–∞—Ç—ã, –Ω–æ "–∫—Ä–µ–¥–∏—Ç" –∑–¥–µ—Å—å –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω –≤ –≤–∏–¥–µ **–ø–æ—Ç–µ–Ω—Ü–∏–∞–ª—å–Ω–æ–π —Ñ—É–Ω–∫—Ü–∏–∏ $\Phi$**.

* –ü–æ—Ç–µ–Ω—Ü–∏–∞–ª—å–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è $\Phi(D)$ —Å–æ–ø–æ—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ—Å—Ç–æ—è–Ω–∏—é —Å—Ç—Ä—É–∫—Ç—É—Ä—ã –¥–∞–Ω–Ω—ã—Ö $D$ –Ω–µ–∫–æ—Ç–æ—Ä–æ–µ —á–∏—Å–ª–æ (–ø–æ—Ç–µ–Ω—Ü–∏–∞–ª).
* –í –Ω–∞—á–∞–ª–µ $\Phi(D_0) = 0$ –∏ $\Phi(D_i) \geq 0$ –¥–ª—è –≤—Å–µ—Ö —Å–æ—Å—Ç–æ—è–Ω–∏–π.
* –ê–º–æ—Ä—Ç–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω–∞—è —Å—Ç–æ–∏–º–æ—Å—Ç—å $a_i$ –æ–ø–µ—Ä–∞—Ü–∏–∏ –≤—ã—á–∏—Å–ª—è–µ—Ç—Å—è –ø–æ —Ñ–æ—Ä–º—É–ª–µ:
    $$a_i = c_i + \Delta\Phi = c_i + (\Phi(D_i) - \Phi(D_{i-1}))$$
    –≥–¥–µ $c_i$ ‚Äî —Ä–µ–∞–ª—å–Ω–∞—è —Å—Ç–æ–∏–º–æ—Å—Ç—å, –∞ $\Delta\Phi$ ‚Äî –∏–∑–º–µ–Ω–µ–Ω–∏–µ –ø–æ—Ç–µ–Ω—Ü–∏–∞–ª–∞ –ø–æ—Å–ª–µ –æ–ø–µ—Ä–∞—Ü–∏–∏.

**–ò–¥–µ—è:**
* "–î–µ—à—ë–≤—ã–µ" –æ–ø–µ—Ä–∞—Ü–∏–∏ –¥–æ–ª–∂–Ω—ã **—É–≤–µ–ª–∏—á–∏–≤–∞—Ç—å** –ø–æ—Ç–µ–Ω—Ü–∏–∞–ª ($\Delta\Phi > 0$).
* "–î–æ—Ä–æ–≥–∏–µ" –æ–ø–µ—Ä–∞—Ü–∏–∏ –¥–æ–ª–∂–Ω—ã **—É–º–µ–Ω—å—à–∞—Ç—å** –ø–æ—Ç–µ–Ω—Ü–∏–∞–ª ($\Delta\Phi < 0$). –ë–æ–ª—å—à–æ–µ –ø–∞–¥–µ–Ω–∏–µ –ø–æ—Ç–µ–Ω—Ü–∏–∞–ª–∞ –∫–æ–º–ø–µ–Ω—Å–∏—Ä—É–µ—Ç –≤—ã—Å–æ–∫—É—é —Ä–µ–∞–ª—å–Ω—É—é —Å—Ç–æ–∏–º–æ—Å—Ç—å $c_i$.

**–ü—Ä–∏–º–µ—Ä: –î–∏–Ω–∞–º–∏—á–µ—Å–∫–∏–π –º–∞—Å—Å–∏–≤.**
* –û–ø—Ä–µ–¥–µ–ª–∏–º –ø–æ—Ç–µ–Ω—Ü–∏–∞–ª –∫–∞–∫: $\Phi(D) = 2 \cdot \text{size} - \text{capacity}$.
* **–°–ª—É—á–∞–π 1: –ú–µ—Å—Ç–æ –µ—Å—Ç—å (–±–µ–∑ —Ä–∞—Å—à–∏—Ä–µ–Ω–∏—è).**
    * `size` —É–≤–µ–ª–∏—á–∏–≤–∞–µ—Ç—Å—è –Ω–∞ 1, `capacity` –Ω–µ –º–µ–Ω—è–µ—Ç—Å—è.
    * –†–µ–∞–ª—å–Ω–∞—è —Å—Ç–æ–∏–º–æ—Å—Ç—å $c_i = 1$.
    * –ò–∑–º–µ–Ω–µ–Ω–∏–µ –ø–æ—Ç–µ–Ω—Ü–∏–∞–ª–∞ $\Delta\Phi = (2(\text{size}+1) - \text{capacity}) - (2\cdot\text{size} - \text{capacity}) = 2$.
    * –ê–º–æ—Ä—Ç–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω–∞—è —Å—Ç–æ–∏–º–æ—Å—Ç—å $a_i = c_i + \Delta\Phi = 1 + 2 = 3$.
* **–°–ª—É—á–∞–π 2: –ú–µ—Å—Ç–æ –∑–∞–∫–æ–Ω—á–∏–ª–æ—Å—å (—Ä–∞—Å—à–∏—Ä–µ–Ω–∏–µ —Å $k$ –¥–æ $2k$ —ç–ª–µ–º–µ–Ω—Ç–æ–≤).**
    * `size` —Å—Ç–∞–Ω–æ–≤–∏—Ç—Å—è $k+1$, `capacity` —Å—Ç–∞–Ω–æ–≤–∏—Ç—Å—è $2k$.
    * –†–µ–∞–ª—å–Ω–∞—è —Å—Ç–æ–∏–º–æ—Å—Ç—å $c_i = k+1$ (–∫–æ–ø–∏—Ä–æ–≤–∞–Ω–∏–µ + –≤—Å—Ç–∞–≤–∫–∞).
    * –ò–∑–º–µ–Ω–µ–Ω–∏–µ –ø–æ—Ç–µ–Ω—Ü–∏–∞–ª–∞ $\Delta\Phi = (2(k+1) - 2k) - (2k - k) = (2) - (k) = 2-k$.
    * –ê–º–æ—Ä—Ç–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω–∞—è —Å—Ç–æ–∏–º–æ—Å—Ç—å $a_i = c_i + \Delta\Phi = (k+1) + (2-k) = 3$.

–í –æ–±–æ–∏—Ö —Å–ª—É—á–∞—è—Ö –∞–º–æ—Ä—Ç–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω–∞—è —Å—Ç–æ–∏–º–æ—Å—Ç—å –ø–æ–ª—É—á–∏–ª–∞—Å—å —Ä–∞–≤–Ω–æ–π 3, —Ç–æ –µ—Å—Ç—å $O(1)$.

### –†–µ–∞–ª–∏–∑–∞—Ü–∏—è –º–∏–Ω–∏–º—É–º–∞ –Ω–∞ —Å—Ç–µ–∫–µ

**–†–µ—à–µ–Ω–∏–µ:** –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å **–¥–≤–∞ —Å—Ç–µ–∫–∞**:

* **`pop()`**:
    1.  –ò–∑–≤–ª–µ–∫–∞–µ–º —ç–ª–µ–º–µ–Ω—Ç `item` –∏–∑ `main_stack`.
    2.  –ï—Å–ª–∏ `item` —Ä–∞–≤–µ–Ω –≤–µ—Ä—à–∏–Ω–µ `min_stack`, –∏–∑–≤–ª–µ–∫–∞–µ–º —ç–ª–µ–º–µ–Ω—Ç –∏ –∏–∑ `min_stack`. –≠—Ç–æ –∑–Ω–∞—á–∏—Ç, —á—Ç–æ –º—ã —É–¥–∞–ª–∏–ª–∏ —Ç–µ–∫—É—â–∏–π –º–∏–Ω–∏–º—É–º, –∏ —Ç–µ–ø–µ—Ä—å –º–∏–Ω–∏–º—É–º ‚Äî —ç—Ç–æ —Å–ª–µ–¥—É—é—â–∏–π —ç–ª–µ–º–µ–Ω—Ç –≤ `min_stack`.
* **`get_min()`**:
    1.  –ü—Ä–æ—Å—Ç–æ –≤–æ–∑–≤—Ä–∞—â–∞–µ–º –≤–µ—Ä—à–∏–Ω—É `min_stack`.

  def __init__(self):
    self.main_stack = []
    self.min_stack = []

  def push(self, item):
    self.main_stack.append(item)
    if not self.min_stack or item <= self.min_stack[-1]:
      self.min_stack.append(item)

  def pop(self):

      return "Error: Stack is empty"

## –ë–∏–ª–µ—Ç 7.

### –û—á–µ—Ä–µ–¥—å: –û—Å–Ω–æ–≤–Ω—ã–µ –ø–æ–Ω—è—Ç–∏—è

**–ò–¥–µ—è:**

**–õ–æ–≥–∏–∫–∞ —Ä–∞–±–æ—Ç—ã:**

    * –ï—Å–ª–∏ `out_stack` **–ø—É—Å—Ç**, –º—ã –¥–æ–ª–∂–Ω—ã "–ø–µ—Ä–µ–ª–∏—Ç—å" –≤ –Ω–µ–≥–æ –≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã –∏–∑ `in_stack`. –ü—Ä–∏ —ç—Ç–æ–º –æ–Ω–∏ –ø–µ—Ä–µ–≤–µ—Ä–Ω—É—Ç—Å—è, –∏ —Å–∞–º—ã–π —Å—Ç–∞—Ä—ã–π —ç–ª–µ–º–µ–Ω—Ç –∏–∑ `in_stack` –æ–∫–∞–∂–µ—Ç—Å—è –Ω–∞ –≤–µ—Ä—à–∏–Ω–µ `out_stack`. –ü–æ—Å–ª–µ —ç—Ç–æ–≥–æ –∏–∑–≤–ª–µ–∫–∞–µ–º —ç–ª–µ–º–µ–Ω—Ç.

–≠—Ç–∞ "–ø–µ—Ä–µ–ª–∏–≤–∞—é—â–∞—è" –æ–ø–µ—Ä–∞—Ü–∏—è –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è —Ç–æ–ª—å–∫–æ —Ç–æ–≥–¥–∞, –∫–æ–≥–¥–∞ `out_stack` –æ–ø—É—Å—Ç–æ—à–∞–µ—Ç—Å—è.

#### –ê–Ω–∞–ª–∏–∑ —Å–ª–æ–∂–Ω–æ—Å—Ç–∏

* **`enqueue`**: –í—Å–µ–≥–¥–∞ –æ–¥–Ω–∞ –æ–ø–µ—Ä–∞—Ü–∏—è `push` –≤ `in_stack`. **–°–ª–æ–∂–Ω–æ—Å—Ç—å: $O(1)$**.
* **`dequeue`**:
    * **–•—É–¥—à–∏–π —Å–ª—É—á–∞–π**: –ï—Å–ª–∏ `out_stack` –ø—É—Å—Ç, –∞ –≤ `in_stack` –Ω–∞—Ö–æ–¥–∏—Ç—Å—è `n` —ç–ª–µ–º–µ–Ω—Ç–æ–≤, –Ω–∞–º –Ω—É–∂–Ω–æ –≤—ã–ø–æ–ª–Ω–∏—Ç—å `n` –æ–ø–µ—Ä–∞—Ü–∏–π `pop` –∏–∑ `in_stack` –∏ `n` –æ–ø–µ—Ä–∞—Ü–∏–π `push` –≤ `out_stack`. **–°–ª–æ–∂–Ω–æ—Å—Ç—å: $O(n)$**.
    * **–õ—É—á—à–∏–π —Å–ª—É—á–∞–π**: –ï—Å–ª–∏ `out_stack` –Ω–µ –ø—É—Å—Ç, –Ω—É–∂–Ω–∞ –≤—Å–µ–≥–æ –æ–¥–Ω–∞ –æ–ø–µ—Ä–∞—Ü–∏—è `pop`. **–°–ª–æ–∂–Ω–æ—Å—Ç—å: $O(1)$**.
    * **–ê–º–æ—Ä—Ç–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω–∞—è —Å–ª–æ–∂–Ω–æ—Å—Ç—å**: –ö–∞–∂–¥—ã–π —ç–ª–µ–º–µ–Ω—Ç –∑–∞ –≤—Å—ë –≤—Ä–µ–º—è —Å–≤–æ–µ–≥–æ —Å—É—â–µ—Å—Ç–≤–æ–≤–∞–Ω–∏—è –≤ –æ—á–µ—Ä–µ–¥–∏ —Ä–æ–≤–Ω–æ –æ–¥–∏–Ω —Ä–∞–∑ –¥–æ–±–∞–≤–ª—è–µ—Ç—Å—è –≤ `in_stack`, –æ–¥–∏–Ω —Ä–∞–∑ –ø–µ—Ä–µ–∫–ª–∞–¥—ã–≤–∞–µ—Ç—Å—è –≤ `out_stack` –∏ –æ–¥–∏–Ω —Ä–∞–∑ –∏–∑–≤–ª–µ–∫–∞–µ—Ç—Å—è –∏–∑ –Ω–µ–≥–æ. –¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, —Å—Ä–µ–¥–Ω—è—è —Å—Ç–æ–∏–º–æ—Å—Ç—å –æ–ø–µ—Ä–∞—Ü–∏–∏ `dequeue` –Ω–∞ –¥–ª–∏–Ω–Ω–æ–π –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ —Å–≤–æ–¥–∏—Ç—Å—è –∫ –∫–æ–Ω—Å—Ç–∞–Ω—Ç–µ. **–ê–º–æ—Ä—Ç–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω–∞—è —Å–ª–æ–∂–Ω–æ—Å—Ç—å: $O(1)$**.

#### –ü—Ä–∏–º–µ—Ä —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏

```python
class QueueWithStacks:
  def __init__(self):
    self.in_stack = []
    self.out_stack = []

  def enqueue(self, item):
    """–î–æ–±–∞–≤–ª—è–µ—Ç —ç–ª–µ–º–µ–Ω—Ç –≤ –æ—á–µ—Ä–µ–¥—å. –°–ª–æ–∂–Ω–æ—Å—Ç—å O(1)."""
    self.in_stack.append(item)

  def _transfer_if_needed(self):
    """
    –í—Å–ø–æ–º–æ–≥–∞—Ç–µ–ª—å–Ω—ã–π –º–µ—Ç–æ–¥. –ï—Å–ª–∏ out_stack –ø—É—Å—Ç,
    –ø–µ—Ä–µ–∫–ª–∞–¥—ã–≤–∞–µ—Ç —ç–ª–µ–º–µ–Ω—Ç—ã –∏–∑ in_stack.
    """
    if not self.out_stack:
      while self.in_stack:
        self.out_stack.append(self.in_stack.pop())

  def dequeue(self):
    """–ò–∑–≤–ª–µ–∫–∞–µ—Ç —ç–ª–µ–º–µ–Ω—Ç –∏–∑ –æ—á–µ—Ä–µ–¥–∏. –ê–º–æ—Ä—Ç–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω–∞—è —Å–ª–æ–∂–Ω–æ—Å—Ç—å O(1)."""
    if self.is_empty():
      return "Error: Queue is empty"
    self._transfer_if_needed()
    return self.out_stack.pop()

  def peek(self):
    """–°–º–æ—Ç—Ä–∏—Ç –Ω–∞ –ø–µ—Ä–≤—ã–π —ç–ª–µ–º–µ–Ω—Ç. –ê–º–æ—Ä—Ç–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω–∞—è —Å–ª–æ–∂–Ω–æ—Å—Ç—å O(1)."""
    if self.is_empty():
      return "Error: Queue is empty"
    self._transfer_if_needed()
    return self.out_stack[-1]

  def is_empty(self):
    return not self.in_stack and not self.out_stack

# –î–µ–º–æ–Ω—Å—Ç—Ä–∞—Ü–∏—è —Ä–∞–±–æ—Ç—ã
q = QueueWithStacks()
q.enqueue(1)
q.enqueue(2)
q.enqueue(3)

print(q.dequeue()) # -> 1. out_stack –±—ã–ª –ø—É—Å—Ç, —ç–ª–µ–º–µ–Ω—Ç—ã [1, 2, 3] –ø–µ—Ä–µ–ª–æ–∂–∏–ª–∏—Å—å
print(q.peek())    # -> 2. out_stack –Ω–µ –ø—É—Å—Ç, –ø—Ä–æ—Å—Ç–æ —Å–º–æ—Ç—Ä–∏–º –Ω–∞ –≤–µ—Ä—à–∏–Ω—É
print(q.dequeue()) # -> 2. out_stack –Ω–µ –ø—É—Å—Ç, –∏–∑–≤–ª–µ–∫–∞–µ–º
```

### 1. –ü—Ä–∏–æ—Ä–∏—Ç–µ—Ç–Ω–∞—è –æ—á–µ—Ä–µ–¥—å

**–ü—Ä–∏–æ—Ä–∏—Ç–µ—Ç–Ω–∞—è –æ—á–µ—Ä–µ–¥—å (Priority Queue)** ‚Äî —ç—Ç–æ –∞–±—Å—Ç—Ä–∞–∫—Ç–Ω—ã–π —Ç–∏–ø –¥–∞–Ω–Ω—ã—Ö, –ø–æ—Ö–æ–∂–∏–π –Ω–∞ –æ–±—ã—á–Ω—É—é –æ—á–µ—Ä–µ–¥—å, –Ω–æ —É –∫–∞–∂–¥–æ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞ –µ—Å—Ç—å **–ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç**. –≠–ª–µ–º–µ–Ω—Ç—ã —Å –±–æ–ª–µ–µ –≤—ã—Å–æ–∫–∏–º –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç–æ–º –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞—é—Ç—Å—è —Ä–∞–Ω—å—à–µ, —á–µ–º —ç–ª–µ–º–µ–Ω—Ç—ã —Å –Ω–∏–∑–∫–∏–º. –ï—Å–ª–∏ –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç—ã —Ä–∞–≤–Ω—ã, –ø–æ—Ä—è–¥–æ–∫ –æ–±—Ä–∞–±–æ—Ç–∫–∏ –æ–±—ã—á–Ω–æ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç FIFO.

### 2. –ë–∏–Ω–∞—Ä–Ω–∞—è –∫—É—á–∞ (Binary Heap)

**–ë–∏–Ω–∞—Ä–Ω–∞—è –∫—É—á–∞** ‚Äî —ç—Ç–æ –Ω–∞–∏–±–æ–ª–µ–µ —Ä–∞—Å–ø—Ä–æ—Å—Ç—Ä–∞–Ω—ë–Ω–Ω—ã–π —Å–ø–æ—Å–æ–± —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç–Ω–æ–π –æ—á–µ—Ä–µ–¥–∏. –≠—Ç–æ **–ø–æ–ª–Ω–æ–µ –±–∏–Ω–∞—Ä–Ω–æ–µ –¥–µ—Ä–µ–≤–æ**, –∫–æ—Ç–æ—Ä–æ–µ —É–¥–æ–≤–ª–µ—Ç–≤–æ—Ä—è–µ—Ç **—Å–≤–æ–π—Å—Ç–≤—É –∫—É—á–∏**.

* **–ü–æ–ª–Ω–æ–µ –±–∏–Ω–∞—Ä–Ω–æ–µ –¥–µ—Ä–µ–≤–æ:** –≤—Å–µ —É—Ä–æ–≤–Ω–∏ –¥–µ—Ä–µ–≤–∞ –ø–æ–ª–Ω–æ—Å—Ç—å—é –∑–∞–ø–æ–ª–Ω–µ–Ω—ã, –∑–∞ –∏—Å–∫–ª—é—á–µ–Ω–∏–µ–º, –≤–æ–∑–º–æ–∂–Ω–æ, –ø–æ—Å–ª–µ–¥–Ω–µ–≥–æ, –∫–æ—Ç–æ—Ä—ã–π –∑–∞–ø–æ–ª–Ω—è–µ—Ç—Å—è —Å–ª–µ–≤–∞ –Ω–∞–ø—Ä–∞–≤–æ. –≠—Ç–æ –ø–æ–∑–≤–æ–ª—è–µ—Ç —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ —Ö—Ä–∞–Ω–∏—Ç—å –∫—É—á—É –≤ –≤–∏–¥–µ –æ–±—ã—á–Ω–æ–≥–æ –º–∞—Å—Å–∏–≤–∞.
* **–°–≤–æ–π—Å—Ç–≤–æ –∫—É—á–∏:**
    * **Min-heap (–º–∏–Ω–∏–º–∞–ª—å–Ω–∞—è –∫—É—á–∞):** –∑–Ω–∞—á–µ–Ω–∏–µ –≤ –ª—é–±–æ–π –≤–µ—Ä—à–∏–Ω–µ *–º–µ–Ω—å—à–µ –∏–ª–∏ —Ä–∞–≤–Ω–æ* –∑–Ω–∞—á–µ–Ω–∏—è–º –≤ –µ—ë –¥–æ—á–µ—Ä–Ω–∏—Ö –≤–µ—Ä—à–∏–Ω–∞—Ö. –°–∞–º—ã–π –º–∞–ª–µ–Ω—å–∫–∏–π —ç–ª–µ–º–µ–Ω—Ç –≤—Å–µ–≥–¥–∞ –Ω–∞—Ö–æ–¥–∏—Ç—Å—è –≤ –∫–æ—Ä–Ω–µ.
    * **Max-heap (–º–∞–∫—Å–∏–º–∞–ª—å–Ω–∞—è –∫—É—á–∞):** –∑–Ω–∞—á–µ–Ω–∏–µ –≤ –ª—é–±–æ–π –≤–µ—Ä—à–∏–Ω–µ *–±–æ–ª—å—à–µ –∏–ª–∏ —Ä–∞–≤–Ω–æ* –∑–Ω–∞—á–µ–Ω–∏—è–º –≤ –µ—ë –¥–æ—á–µ—Ä–Ω–∏—Ö –≤–µ—Ä—à–∏–Ω–∞—Ö. –°–∞–º—ã–π –±–æ–ª—å—à–æ–π —ç–ª–µ–º–µ–Ω—Ç –≤—Å–µ–≥–¥–∞ –≤ –∫–æ—Ä–Ω–µ.

#### –ò–Ω—Ç–µ—Ä—Ñ–µ–π—Å –∏ —Å–ª–æ–∂–Ω–æ—Å—Ç—å

| –û–ø–µ—Ä–∞—Ü–∏—è | –û–ø–∏—Å–∞–Ω–∏–µ | –°–ª–æ–∂–Ω–æ—Å—Ç—å |

# heapq —Ä–∞–±–æ—Ç–∞–µ—Ç —Å–æ —Å–ø–∏—Å–∫–æ–º Python, –ø—Ä–µ–≤—Ä–∞—â–∞—è –µ–≥–æ –≤ min-heap

### 3. –°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –∫—É—á–µ–π (Heapsort)

**–°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –∫—É—á–µ–π** ‚Äî —ç—Ç–æ —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω—ã–π –∞–ª–≥–æ—Ä–∏—Ç–º —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∏, –∏—Å–ø–æ–ª—å–∑—É—é—â–∏–π —Å—Ç—Ä—É–∫—Ç—É—Ä—É –±–∏–Ω–∞—Ä–Ω–æ–π –∫—É—á–∏.

**–ê–ª–≥–æ—Ä–∏—Ç–º (–¥–ª—è —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∏ –ø–æ –≤–æ–∑—Ä–∞—Å—Ç–∞–Ω–∏—é):**
1.  **–ü–æ—Å—Ç—Ä–æ–µ–Ω–∏–µ –∫—É—á–∏ (Heapify):** –ü—Ä–µ–æ–±—Ä–∞–∑—É–µ–º –∏—Å—Ö–æ–¥–Ω—ã–π –º–∞—Å—Å–∏–≤ –≤ **max-heap**. –°–∞–º—ã–π –±–æ–ª—å—à–æ–π —ç–ª–µ–º–µ–Ω—Ç –æ–∫–∞–∂–µ—Ç—Å—è –≤ –∫–æ—Ä–Ω–µ. –≠—Ç–æ –º–æ–∂–Ω–æ —Å–¥–µ–ª–∞—Ç—å –∑–∞ $O(n)$.
2.  **–°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞:**
    * –ú–µ–Ω—è–µ–º –º–µ—Å—Ç–∞–º–∏ –∫–æ—Ä–Ω–µ–≤–æ–π (–º–∞–∫—Å–∏–º–∞–ª—å–Ω—ã–π) —ç–ª–µ–º–µ–Ω—Ç —Å –ø–æ—Å–ª–µ–¥–Ω–∏–º —ç–ª–µ–º–µ–Ω—Ç–æ–º –≤ –∫—É—á–µ.
    * "–ò—Å–∫–ª—é—á–∞–µ–º" —ç—Ç–æ—Ç –ø–æ—Å–ª–µ–¥–Ω–∏–π —ç–ª–µ–º–µ–Ω—Ç –∏–∑ —Ä–∞—Å—Å–º–æ—Ç—Ä–µ–Ω–∏—è ‚Äî —Ç–µ–ø–µ—Ä—å –æ–Ω –Ω–∞ —Å–≤–æ—ë–º –æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–º –º–µ—Å—Ç–µ.
    * –í–æ—Å—Å—Ç–∞–Ω–∞–≤–ª–∏–≤–∞–µ–º —Å–≤–æ–π—Å—Ç–≤–æ max-heap –¥–ª—è –æ—Å—Ç–∞–≤—à–µ–π—Å—è —á–∞—Å—Ç–∏, "–ø—Ä–æ—Å–µ–∏–≤–∞—è –≤–Ω–∏–∑" –Ω–æ–≤—ã–π –∫–æ—Ä–µ–Ω—å.
    * –ü–æ–≤—Ç–æ—Ä—è–µ–º –ø—Ä–æ—Ü–µ—Å—Å, –ø–æ–∫–∞ –≤ –∫—É—á–µ –Ω–µ –æ—Å—Ç–∞–Ω–µ—Ç—Å—è –æ–¥–∏–Ω —ç–ª–µ–º–µ–Ω—Ç.

**–°–ª–æ–∂–Ω–æ—Å—Ç—å:**
* **–í—Ä–µ–º–µ–Ω–Ω–∞—è:** $O(n \log n)$ –¥–ª—è —Ö—É–¥—à–µ–≥–æ, —Å—Ä–µ–¥–Ω–µ–≥–æ –∏ –ª—É—á—à–µ–≥–æ —Å–ª—É—á–∞–µ–≤.
* **–ü—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–µ–Ω–Ω–∞—è:** $O(1)$, —Ç–∞–∫ –∫–∞–∫ —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç –Ω–∞ –º–µ—Å—Ç–µ (in-place).

```python
def heapify(arr, n, i):
    """–ü—Ä–µ–≤—Ä–∞—â–∞–µ—Ç –ø–æ–¥–¥–µ—Ä–µ–≤–æ —Å –∫–æ—Ä–Ω–µ–º i –≤ max-heap."""
    largest = i
    left = 2 * i + 1
    right = 2 * i + 2

    if left < n and arr[left] > arr[largest]:
        largest = left
    if right < n and arr[right] > arr[largest]:
        largest = right
    if largest != i:
        arr[i], arr[largest] = arr[largest], arr[i]
        heapify(arr, n, largest)

def heap_sort(arr):
    n = len(arr)
    # 1. –°—Ç—Ä–æ–∏–º max-heap
    for i in range(n // 2 - 1, -1, -1):
        heapify(arr, n, i)
    
    # 2. –ò–∑–≤–ª–µ–∫–∞–µ–º —ç–ª–µ–º–µ–Ω—Ç—ã –æ–¥–∏–Ω –∑–∞ –¥—Ä—É–≥–∏–º
    for i in range(n - 1, 0, -1):
        arr[i], arr[0] = arr[0], arr[i]  # –ü–µ—Ä–µ–º–µ—â–∞–µ–º –∫–æ—Ä–µ–Ω—å –≤ –∫–æ–Ω–µ—Ü
        heapify(arr, i, 0) # –í–æ—Å—Å—Ç–∞–Ω–∞–≤–ª–∏–≤–∞–µ–º –∫—É—á—É –¥–ª—è –æ—Å—Ç–∞–≤—à–∏—Ö—Å—è —ç–ª–µ–º–µ–Ω—Ç–æ–≤
    return arr

my_array = [12, 11, 13, 5, 6, 7]
print(f"–û—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω—ã–π –º–∞—Å—Å–∏–≤: {heap_sort(my_array)}")
# -> –û—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω—ã–π –º–∞—Å—Å–∏–≤: [5, 6, 7, 11, 12, 13]
```

### 3\. –ú–∞—Ç—Ä–∏—Ü–∞ –∏–Ω—Ü–∏–¥–µ–Ω—Ç–Ω–æ—Å—Ç–∏ (Incidence Matrix)

–≠—Ç–æ –º–∞—Ç—Ä–∏—Ü–∞ —Ä–∞–∑–º–µ—Ä–æ–º $|V| \\times |E|$. –≠–ª–µ–º–µ–Ω—Ç `M[i][j]` —Ä–∞–≤–µ–Ω 1, –µ—Å–ª–∏ –≤–µ—Ä—à–∏–Ω–∞ `i` —è–≤–ª—è–µ—Ç—Å—è –∫–æ–Ω—Ü–æ–º —Ä–µ–±—Ä–∞ `j`.

  * **–ü—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–µ:** –î–≤—É–º–µ—Ä–Ω—ã–π –º–∞—Å—Å–∏–≤.
  * **–ü–ª—é—Å—ã:** –£–¥–æ–±–Ω–∞ –¥–ª—è –Ω–µ–∫–æ—Ç–æ—Ä—ã—Ö —Ç–µ–æ—Ä–µ—Ç–∏—á–µ—Å–∫–∏—Ö –∑–∞–¥–∞—á –∏ –∞–ª–≥–æ—Ä–∏—Ç–º–æ–≤, —Å–≤—è–∑–∞–Ω–Ω—ã—Ö —Å –ø–æ—Ç–æ–∫–∞–º–∏.
  * **–ú–∏–Ω—É—Å—ã:** –†–µ–¥–∫–æ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –Ω–∞ –ø—Ä–∞–∫—Ç–∏–∫–µ –¥–ª—è –æ–±—â–∏—Ö –∑–∞–¥–∞—á (–æ–±—Ö–æ–¥, –ø–æ–∏—Å–∫ –ø—É—Ç–µ–π), —Ç–∞–∫ –∫–∞–∫ —Ç—Ä–µ–±—É–µ—Ç $O(V \\cdot E)$ –ø–∞–º—è—Ç–∏.

# –î–ª—è –Ω–∞—à–µ–≥–æ –ø—Ä–∏–º–µ—Ä–∞ –≥—Ä–∞—Ñ–∞
# –†—ë–±—Ä–∞: e0=(0,1), e1=(0,2), e2=(1,2), e3=(1,3)

## –û–±—Ö–æ–¥ –≥—Ä–∞—Ñ–∞ –≤ —à–∏—Ä–∏–Ω—É (BFS, Breadth-First Search)

**–û–±—Ö–æ–¥ –≤ —à–∏—Ä–∏–Ω—É** ‚Äî —ç—Ç–æ –∞–ª–≥–æ—Ä–∏—Ç–º –æ–±—Ö–æ–¥–∞ –≥—Ä–∞—Ñ–∞, –∫–æ—Ç–æ—Ä—ã–π –∏—Å—Å–ª–µ–¥—É–µ—Ç –≤–µ—Ä—à–∏–Ω—ã "—Å–ª–æ–π –∑–∞ —Å–ª–æ–µ–º". –û–Ω –Ω–∞—á–∏–Ω–∞–µ—Ç —Å –∏—Å—Ö–æ–¥–Ω–æ–π –≤–µ—Ä—à–∏–Ω—ã, –ø–æ—Å–µ—â–∞–µ—Ç –≤—Å–µ—Ö –µ—ë —Å–æ—Å–µ–¥–µ–π, –∑–∞—Ç–µ–º —Å–æ—Å–µ–¥–µ–π —ç—Ç–∏—Ö —Å–æ—Å–µ–¥–µ–π –∏ —Ç–∞–∫ –¥–∞–ª–µ–µ. –î–ª—è —Å–≤–æ–µ–π —Ä–∞–±–æ—Ç—ã BFS –∏—Å–ø–æ–ª—å–∑—É–µ—Ç **–æ—á–µ—Ä–µ–¥—å (Queue)** FIFO.

**–û—Å–Ω–æ–≤–Ω–æ–µ –ø—Ä–∏–º–µ–Ω–µ–Ω–∏–µ:** –ø–æ–∏—Å–∫ –∫—Ä–∞—Ç—á–∞–π—à–µ–≥–æ –ø—É—Ç–∏ –≤ **–Ω–µ–≤–∑–≤–µ—à–µ–Ω–Ω–æ–º** –≥—Ä–∞—Ñ–µ.

### –ê–ª–≥–æ—Ä–∏—Ç–º BFS

1.  –°–æ–∑–¥–∞—Ç—å **–æ—á–µ—Ä–µ–¥—å** –∏ –¥–æ–±–∞–≤–∏—Ç—å –≤ –Ω–µ—ë —Å—Ç–∞—Ä—Ç–æ–≤—É—é –≤–µ—Ä—à–∏–Ω—É.
2.  –°–æ–∑–¥–∞—Ç—å –º–Ω–æ–∂–µ—Å—Ç–≤–æ `visited` –¥–ª—è –æ—Ç—Å–ª–µ–∂–∏–≤–∞–Ω–∏—è –ø–æ—Å–µ—â—ë–Ω–Ω—ã—Ö –≤–µ—Ä—à–∏–Ω –∏ –¥–æ–±–∞–≤–∏—Ç—å –≤ –Ω–µ–≥–æ —Å—Ç–∞—Ä—Ç–æ–≤—É—é –≤–µ—Ä—à–∏–Ω—É.
3.  –ü–æ–∫–∞ –æ—á–µ—Ä–µ–¥—å –Ω–µ –ø—É—Å—Ç–∞:
    a. –ò–∑–≤–ª–µ—á—å –≤–µ—Ä—à–∏–Ω—É `u` –∏–∑ –Ω–∞—á–∞–ª–∞ –æ—á–µ—Ä–µ–¥–∏.
    b. –û–±—Ä–∞–±–æ—Ç–∞—Ç—å `u` (–Ω–∞–ø—Ä–∏–º–µ—Ä, –≤—ã–≤–µ—Å—Ç–∏ –Ω–∞ —ç–∫—Ä–∞–Ω).
    c. –î–ª—è –∫–∞–∂–¥–æ–≥–æ —Å–æ—Å–µ–¥–∞ `v` –≤–µ—Ä—à–∏–Ω—ã `u`:
    i. –ï—Å–ª–∏ `v` –µ—â—ë –Ω–µ –ø–æ—Å–µ—â–∞–ª–∞—Å—å, –¥–æ–±–∞–≤–∏—Ç—å `v` –≤ `visited` –∏ –≤ –æ—á–µ—Ä–µ–¥—å.

### –ü—Ä–∏–º–µ—Ä –Ω–∞ Python

```python
from collections import deque

def bfs(graph, start_node):
    """–û–±—Ö–æ–¥ –≥—Ä–∞—Ñ–∞ –≤ —à–∏—Ä–∏–Ω—É."""
    visited = set()         # –ú–Ω–æ–∂–µ—Å—Ç–≤–æ –¥–ª—è –ø–æ—Å–µ—â—ë–Ω–Ω—ã—Ö –≤–µ—Ä—à–∏–Ω
    queue = deque([start_node]) # –û—á–µ—Ä–µ–¥—å –¥–ª—è –≤–µ—Ä—à–∏–Ω –∫ –ø–æ—Å–µ—â–µ–Ω–∏—é
    visited.add(start_node)
    
    result = [] # –°–ø–∏—Å–æ–∫ –¥–ª—è —Ö—Ä–∞–Ω–µ–Ω–∏—è –ø–æ—Ä—è–¥–∫–∞ –æ–±—Ö–æ–¥–∞
    
    while queue:
        vertex = queue.popleft() # –ò–∑–≤–ª–µ–∫–∞–µ–º –≤–µ—Ä—à–∏–Ω—É –∏–∑ –Ω–∞—á–∞–ª–∞ –æ—á–µ—Ä–µ–¥–∏
        result.append(vertex)
        
        for neighbor in graph[vertex]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)
    return result

# –ò—Å–ø–æ–ª—å–∑—É–µ–º –≥—Ä–∞—Ñ –∏–∑ –ø—Ä–∏–º–µ—Ä–∞ –≤—ã—à–µ
graph_example = {
    0: [1, 2],
    1: [0, 2, 3],
    2: [0, 1],
    3: [1]
}

# –ù–∞—á–∏–Ω–∞–µ–º –æ–±—Ö–æ–¥ —Å –≤–µ—Ä—à–∏–Ω—ã 0
bfs_path = bfs(graph_example, 0)
print(f"–ü–æ—Ä—è–¥–æ–∫ –æ–±—Ö–æ–¥–∞ –≤ —à–∏—Ä–∏–Ω—É, –Ω–∞—á–∏–Ω–∞—è —Å 0: {bfs_path}")
# -> –ü–æ—Ä—è–¥–æ–∫ –æ–±—Ö–æ–¥–∞ –≤ —à–∏—Ä–∏–Ω—É, –Ω–∞—á–∏–Ω–∞—è —Å 0: [0, 1, 2, 3]
```

## –û–±—Ö–æ–¥ –≥—Ä–∞—Ñ–∞ –≤ –≥–ª—É–±–∏–Ω—É (DFS, Depth-First Search)

**–û–±—Ö–æ–¥ –≤ –≥–ª—É–±–∏–Ω—É** ‚Äî —ç—Ç–æ –∞–ª–≥–æ—Ä–∏—Ç–º, –∫–æ—Ç–æ—Ä—ã–π –∏—Å—Å–ª–µ–¥—É–µ—Ç –≥—Ä–∞—Ñ, —É—Ö–æ–¥—è "–≤–≥–ª—É–±—å" –ø–æ –æ–¥–Ω–æ–º—É –ø—É—Ç–∏ –Ω–∞—Å—Ç–æ–ª—å–∫–æ, –Ω–∞—Å–∫–æ–ª—å–∫–æ —ç—Ç–æ –≤–æ–∑–º–æ–∂–Ω–æ, –ø—Ä–µ–∂–¥–µ —á–µ–º –≤–µ—Ä–Ω—É—Ç—å—Å—è –Ω–∞–∑–∞–¥ (backtracking). –î–ª—è —Å–≤–æ–µ–π —Ä–∞–±–æ—Ç—ã DFS –∏—Å–ø–æ–ª—å–∑—É–µ—Ç **—Å—Ç–µ–∫ (Stack)** LIFO, –∫–æ—Ç–æ—Ä—ã–π –º–æ–∂–µ—Ç –±—ã—Ç—å –Ω–µ—è–≤–Ω—ã–º (—Å–∏—Å—Ç–µ–º–Ω—ã–π —Å—Ç–µ–∫ –≤—ã–∑–æ–≤–æ–≤ –ø—Ä–∏ —Ä–µ–∫—É—Ä—Å–∏–∏) –∏–ª–∏ —è–≤–Ω—ã–º.

**–û—Å–Ω–æ–≤–Ω—ã–µ –ø—Ä–∏–º–µ–Ω–µ–Ω–∏—è:** –ø–æ–∏—Å–∫ —Ü–∏–∫–ª–æ–≤, —Ç–æ–ø–æ–ª–æ–≥–∏—á–µ—Å–∫–∞—è —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞, –ø–æ–∏—Å–∫ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç —Å–≤—è–∑–Ω–æ—Å—Ç–∏.

### –ê–ª–≥–æ—Ä–∏—Ç–º DFS (—Ä–µ–∫—É—Ä—Å–∏–≤–Ω—ã–π)

1.  –û—Ç–º–µ—Ç–∏—Ç—å —Ç–µ–∫—É—â—É—é –≤–µ—Ä—à–∏–Ω—É `u` –∫–∞–∫ –ø–æ—Å–µ—â—ë–Ω–Ω—É—é.
2.  –û–±—Ä–∞–±–æ—Ç–∞—Ç—å –≤–µ—Ä—à–∏–Ω—É `u`.
3.  –î–ª—è –∫–∞–∂–¥–æ–≥–æ —Å–æ—Å–µ–¥–∞ `v` –≤–µ—Ä—à–∏–Ω—ã `u`:
      * –ï—Å–ª–∏ `v` –µ—â—ë –Ω–µ –ø–æ—Å–µ—â–∞–ª–∞—Å—å, —Ä–µ–∫—É—Ä—Å–∏–≤–Ω–æ –≤—ã–∑–≤–∞—Ç—å DFS –¥–ª—è `v`.

### –ü—Ä–∏–º–µ—Ä –Ω–∞ Python

```python
def dfs(graph, start_node, visited=None):
    """–†–µ–∫—É—Ä—Å–∏–≤–Ω—ã–π –æ–±—Ö–æ–¥ –≥—Ä–∞—Ñ–∞ –≤ –≥–ª—É–±–∏–Ω—É."""
    if visited is None:
        visited = set()
    
    visited.add(start_node)
    print(start_node, end=' ') # –û–±—Ä–∞–±–∞—Ç—ã–≤–∞–µ–º –≤–µ—Ä—à–∏–Ω—É
    
    for neighbor in graph[start_node]:
        if neighbor not in visited:
            dfs(graph, neighbor, visited)

# –ò—Å–ø–æ–ª—å–∑—É–µ–º –≥—Ä–∞—Ñ –∏–∑ –ø—Ä–∏–º–µ—Ä–∞

    0: [1, 2],
    1: [0, 2, 3],
    2: [0, 1],
    3: [1]
}

print("–ü–æ—Ä—è–¥–æ–∫ –æ–±—Ö–æ–¥–∞ –≤ –≥–ª—É–±–∏–Ω—É, –Ω–∞—á–∏–Ω–∞—è —Å 0:", end=' ')
dfs(graph_example, 0)
# -> –ü–æ—Ä—è–¥–æ–∫ –æ–±—Ö–æ–¥–∞ –≤ –≥–ª—É–±–∏–Ω—É, –Ω–∞—á–∏–Ω–∞—è —Å 0: 0 1 2 3 
# (–î—Ä—É–≥–æ–π –≤–æ–∑–º–æ–∂–Ω—ã–π –ø–æ—Ä—è–¥–æ–∫: 0 2 1 3)
```

## –°–≤—è–∑–Ω–æ—Å—Ç—å –≤ –≥—Ä–∞—Ñ–∞—Ö

### –ù–µ–æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ –≥—Ä–∞—Ñ—ã

  * **–°–≤—è–∑–Ω—ã–π –≥—Ä–∞—Ñ (Connected Graph):** –ì—Ä–∞—Ñ –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è —Å–≤—è–∑–Ω—ã–º, –µ—Å–ª–∏ –º–µ–∂–¥—É –ª—é–±—ã–º–∏ –¥–≤—É–º—è –µ–≥–æ –≤–µ—Ä—à–∏–Ω–∞–º–∏ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –ø—É—Ç—å.
  * **–ö–æ–º–ø–æ–Ω–µ–Ω—Ç—ã —Å–≤—è–∑–Ω–æ—Å—Ç–∏ (Connected Components):** –ï—Å–ª–∏ –≥—Ä–∞—Ñ –Ω–µ —è–≤–ª—è–µ—Ç—Å—è —Å–≤—è–∑–Ω—ã–º, –æ–Ω —Ä–∞—Å–ø–∞–¥–∞–µ—Ç—Å—è –Ω–∞ –Ω–µ—Å–∫–æ–ª—å–∫–æ **–∫–æ–º–ø–æ–Ω–µ–Ω—Ç —Å–≤—è–∑–Ω–æ—Å—Ç–∏**. –ö–∞–∂–¥–∞—è —Ç–∞–∫–∞—è –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞ ‚Äî —ç—Ç–æ –º–∞–∫—Å–∏–º–∞–ª—å–Ω—ã–π –ø–æ–¥–≥—Ä–∞—Ñ, –≤ –∫–æ—Ç–æ—Ä–æ–º –≤—Å–µ –≤–µ—Ä—à–∏–Ω—ã —Å–≤—è–∑–∞–Ω—ã –¥—Ä—É–≥ —Å –¥—Ä—É–≥–æ–º. –ù–∞–π—Ç–∏ –≤—Å–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã –º–æ–∂–Ω–æ, –∑–∞–ø—É—Å–∫–∞—è DFS –∏–ª–∏ BFS –∏–∑ –∫–∞–∂–¥–æ–π –µ—â—ë –Ω–µ –ø–æ—Å–µ—â—ë–Ω–Ω–æ–π –≤–µ—Ä—à–∏–Ω—ã.

### –û—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ –≥—Ä–∞—Ñ—ã (Digraphs)

–î–ª—è –æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –≥—Ä–∞—Ñ–æ–≤ –ø–æ–Ω—è—Ç–∏–µ —Å–≤—è–∑–Ω–æ—Å—Ç–∏ —Å–ª–æ–∂–Ω–µ–µ.

  * **–°–ª–∞–±–∞—è —Å–≤—è–∑–Ω–æ—Å—Ç—å (Weakly Connected):** –ì—Ä–∞—Ñ —è–≤–ª—è–µ—Ç—Å—è —Å–ª–∞–±–æ—Å–≤—è–∑–Ω—ã–º, –µ—Å–ª–∏ –æ–Ω —Å—Ç–∞–Ω–æ–≤–∏—Ç—Å—è —Å–≤—è–∑–Ω—ã–º –ø–æ—Å–ª–µ —Ç–æ–≥–æ, –∫–∞–∫ –≤—Å–µ –µ–≥–æ –æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ —Ä—ë–±—Ä–∞ –∑–∞–º–µ–Ω—è—é—Ç—Å—è –Ω–∞ –Ω–µ–æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ. –¢–æ –µ—Å—Ç—å, –º–æ–∂–Ω–æ –¥–æ–±—Ä–∞—Ç—å—Å—è –∏–∑ –ª—é–±–æ–π –≤–µ—Ä—à–∏–Ω—ã –≤ –ª—é–±—É—é, –µ—Å–ª–∏ –∏–≥–Ω–æ—Ä–∏—Ä–æ–≤–∞—Ç—å –Ω–∞–ø—Ä–∞–≤–ª–µ–Ω–∏—è —Å—Ç—Ä–µ–ª–æ–∫.
  * **–°–∏–ª—å–Ω–∞—è —Å–≤—è–∑–Ω–æ—Å—Ç—å (Strongly Connected):** –ì—Ä–∞—Ñ —è–≤–ª—è–µ—Ç—Å—è —Å–∏–ª—å–Ω–æ—Å–≤—è–∑–Ω—ã–º, –µ—Å–ª–∏ –¥–ª—è –ª—é–±–æ–π –ø–∞—Ä—ã –≤–µ—Ä—à–∏–Ω `(u, v)` —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –ø—É—Ç—å –∏–∑ `u` –≤ `v` **–∏** –ø—É—Ç—å –∏–∑ `v` –≤ `u`.
  * **–ö–æ–º–ø–æ–Ω–µ–Ω—Ç—ã —Å–∏–ª—å–Ω–æ–π —Å–≤—è–∑–Ω–æ—Å—Ç–∏ (Strongly Connected Components, SCC):** –≠—Ç–æ –º–∞–∫—Å–∏–º–∞–ª—å–Ω—ã–µ —Å–∏–ª—å–Ω–æ—Å–≤—è–∑–Ω—ã–µ –ø–æ–¥–≥—Ä–∞—Ñ—ã. –ù–∞–π—Ç–∏ –∏—Ö –º–æ–∂–Ω–æ —Å –ø–æ–º–æ—â—å—é —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã—Ö –∞–ª–≥–æ—Ä–∏—Ç–º–æ–≤, —Ç–∞–∫–∏—Ö –∫–∞–∫ **–∞–ª–≥–æ—Ä–∏—Ç–º –¢–∞—Ä—å—è–Ω–∞** –∏–ª–∏ **–∞–ª–≥–æ—Ä–∏—Ç–º –ö–æ—Å–∞—Ä–∞–π—é**.

## –ë–∏–ª–µ—Ç 11. 
–ü–æ–∏—Å–∫ –∫—Ä–∞—Ç—á–∞–π—à–∏—Ö –ø—É—Ç–µ–π –≤–æ –≤–∑–≤–µ—à–µ–Ω–Ω—ã—Ö –≥—Ä–∞—Ñ–∞—Ö. –ê–ª–≥–æ—Ä–∏—Ç–º –î–µ–π–∫—Å—Ç—Ä—ã.

## –ü–æ–∏—Å–∫ –∫—Ä–∞—Ç—á–∞–π—à–∏—Ö –ø—É—Ç–µ–π: –ê–ª–≥–æ—Ä–∏—Ç–º –î–µ–π–∫—Å—Ç—Ä—ã

**–ê–ª–≥–æ—Ä–∏—Ç–º –î–µ–π–∫—Å—Ç—Ä—ã (Dijkstra's Algorithm)** ‚Äî —ç—Ç–æ "–∂–∞–¥–Ω—ã–π" –∞–ª–≥–æ—Ä–∏—Ç–º –¥–ª—è –Ω–∞—Ö–æ–∂–¥–µ–Ω–∏—è –∫—Ä–∞—Ç—á–∞–π—à–∏—Ö –ø—É—Ç–µ–π –æ—Ç –æ–¥–Ω–æ–π —Å—Ç–∞—Ä—Ç–æ–≤–æ–π –≤–µ—Ä—à–∏–Ω—ã –¥–æ –≤—Å–µ—Ö –æ—Å—Ç–∞–ª—å–Ω—ã—Ö –≤–µ—Ä—à–∏–Ω –≤–æ –≤–∑–≤–µ—à–µ–Ω–Ω–æ–º –≥—Ä–∞—Ñ–µ.

**–í–∞–∂–Ω–µ–π—à–µ–µ —É—Å–ª–æ–≤–∏–µ:** –∞–ª–≥–æ—Ä–∏—Ç–º –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ —Ä–∞–±–æ—Ç–∞–µ—Ç —Ç–æ–ª—å–∫–æ –¥–ª—è –≥—Ä–∞—Ñ–æ–≤, –≥–¥–µ **–≤—Å–µ –≤–µ—Å–∞ —Ä—ë–±–µ—Ä –Ω–µ–æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–µ** (‚â• 0).

---

### –û—Å–Ω–æ–≤–Ω–∞—è –∏–¥–µ—è

–ê–ª–≥–æ—Ä–∏—Ç–º –¥–µ–ª–∏—Ç –≤—Å–µ –≤–µ—Ä—à–∏–Ω—ã –Ω–∞ –¥–≤–∞ –º–Ω–æ–∂–µ—Å—Ç–≤–∞:
1.  –í–µ—Ä—à–∏–Ω—ã, –¥–ª—è –∫–æ—Ç–æ—Ä—ã—Ö –∫—Ä–∞—Ç—á–∞–π—à–∏–π –ø—É—Ç—å —É–∂–µ **—Ç–æ—á–Ω–æ –Ω–∞–π–¥–µ–Ω**.
2.  –í–µ—Ä—à–∏–Ω—ã, –¥–ª—è –∫–æ—Ç–æ—Ä—ã—Ö –∫—Ä–∞—Ç—á–∞–π—à–∏–π –ø—É—Ç—å –µ—â—ë **–Ω–µ –Ω–∞–π–¥–µ–Ω** (–∏–ª–∏ —è–≤–ª—è–µ—Ç—Å—è –ø—Ä–µ–¥–≤–∞—Ä–∏—Ç–µ–ª—å–Ω—ã–º).

–ù–∞ –∫–∞–∂–¥–æ–º —à–∞–≥–µ –∞–ª–≥–æ—Ä–∏—Ç–º –≤—ã–±–∏—Ä–∞–µ—Ç "–±–ª–∏–∂–∞–π—à—É—é" –≤–µ—Ä—à–∏–Ω—É –∏–∑ –≤—Ç–æ—Ä–æ–≥–æ –º–Ω–æ–∂–µ—Å—Ç–≤–∞, –¥–ª—è –∫–æ—Ç–æ—Ä–æ–π —Ç–µ–∫—É—â–µ–µ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ –º–∏–Ω–∏–º–∞–ª—å–Ω–æ, –∏ –ø–µ—Ä–µ–Ω–æ—Å–∏—Ç –µ—ë –≤ –ø–µ—Ä–≤–æ–µ. –ó–∞—Ç–µ–º –æ–Ω –æ–±–Ω–æ–≤–ª—è–µ—Ç —Ä–∞—Å—Å—Ç–æ—è–Ω–∏—è –¥–æ –µ—ë —Å–æ—Å–µ–¥–µ–π. –≠—Ç–æ—Ç –ø—Ä–æ—Ü–µ—Å—Å –ø—Ä–æ–¥–æ–ª–∂–∞–µ—Ç—Å—è, –ø–æ–∫–∞ –Ω–µ –±—É–¥—É—Ç –Ω–∞–π–¥–µ–Ω—ã –∫—Ä–∞—Ç—á–∞–π—à–∏–µ –ø—É—Ç–∏ –¥–æ –≤—Å–µ—Ö –¥–æ—Å—Ç–∏–∂–∏–º—ã—Ö –≤–µ—Ä—à–∏–Ω.

–î–ª—è —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ–≥–æ –ø–æ–∏—Å–∫–∞ "–±–ª–∏–∂–∞–π—à–µ–π" –≤–µ—Ä—à–∏–Ω—ã –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è **–ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç–Ω–∞—è –æ—á–µ—Ä–µ–¥—å**.

---

### –ê–ª–≥–æ—Ä–∏—Ç–º

1.  **–ò–Ω–∏—Ü–∏–∞–ª–∏–∑–∞—Ü–∏—è:**
    * –°–æ–∑–¥–∞—Ç—å —Å–ª–æ–≤–∞—Ä—å `distances` –¥–ª—è —Ö—Ä–∞–Ω–µ–Ω–∏—è –∫—Ä–∞—Ç—á–∞–π—à–∏—Ö —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–π. –£—Å—Ç–∞–Ω–æ–≤–∏—Ç—å —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ –¥–æ —Å—Ç–∞—Ä—Ç–æ–≤–æ–π –≤–µ—Ä—à–∏–Ω—ã —Ä–∞–≤–Ω—ã–º 0, –∞ –¥–æ –≤—Å–µ—Ö –æ—Å—Ç–∞–ª—å–Ω—ã—Ö ‚Äî –±–µ—Å–∫–æ–Ω–µ—á–Ω–æ—Å—Ç–∏ ($\infty$).
    * –°–æ–∑–¥–∞—Ç—å **–ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç–Ω—É—é –æ—á–µ—Ä–µ–¥—å** –∏ –¥–æ–±–∞–≤–∏—Ç—å –≤ –Ω–µ—ë —Å—Ç–∞—Ä—Ç–æ–≤—É—é –≤–µ—Ä—à–∏–Ω—É —Å –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç–æ–º 0.

2.  **–û—Å–Ω–æ–≤–Ω–æ–π —Ü–∏–∫–ª:**
    * –ü–æ–∫–∞ –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç–Ω–∞—è –æ—á–µ—Ä–µ–¥—å –Ω–µ –ø—É—Å—Ç–∞:
        a. –ò–∑–≤–ª–µ—á—å –∏–∑ –æ—á–µ—Ä–µ–¥–∏ –≤–µ—Ä—à–∏–Ω—É `u` —Å –Ω–∞–∏–º–µ–Ω—å—à–∏–º —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ–º (–Ω–∞–∏–≤—ã—Å—à–∏–º –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç–æ–º).
        b. –î–ª—è –∫–∞–∂–¥–æ–≥–æ —Å–æ—Å–µ–¥–∞ `v` –≤–µ—Ä—à–∏–Ω—ã `u` –≤—ã–ø–æ–ª–Ω–∏—Ç—å **—Ä–µ–ª–∞–∫—Å–∞—Ü–∏—é —Ä–µ–±—Ä–∞**:
            * –í—ã—á–∏—Å–ª–∏—Ç—å –Ω–æ–≤–æ–µ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ –¥–æ `v` —á–µ—Ä–µ–∑ `u`: `new_distance = distances[u] + weight(u, v)`.
            * –ï—Å–ª–∏ `new_distance` –º–µ–Ω—å—à–µ, —á–µ–º —Ç–µ–∫—É—â–µ–µ –∑–Ω–∞—á–µ–Ω–∏–µ `distances[v]`, —Ç–æ –æ–±–Ω–æ–≤–∏—Ç—å `distances[v] = new_distance` –∏ –¥–æ–±–∞–≤–∏—Ç—å `v` –≤ –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç–Ω—É—é –æ—á–µ—Ä–µ–¥—å —Å –Ω–æ–≤—ã–º, –±–æ–ª–µ–µ –≤—ã—Å–æ–∫–∏–º –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç–æ–º.

3.  **–ó–∞–≤–µ—Ä—à–µ–Ω–∏–µ:**
    * –ö–æ–≥–¥–∞ –æ—á–µ—Ä–µ–¥—å –æ–ø—É—Å—Ç–µ–µ—Ç, –≤ —Å–ª–æ–≤–∞—Ä–µ `distances` –±—É–¥—É—Ç —Ö—Ä–∞–Ω–∏—Ç—å—Å—è –∫—Ä–∞—Ç—á–∞–π—à–∏–µ –ø—É—Ç–∏ –æ—Ç —Å—Ç–∞—Ä—Ç–æ–≤–æ–π –≤–µ—Ä—à–∏–Ω—ã –¥–æ –≤—Å–µ—Ö –æ—Å—Ç–∞–ª—å–Ω—ã—Ö.

### –°–ª–æ–∂–Ω–æ—Å—Ç—å

–ü—Ä–∏ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–∏ –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç–Ω–æ–π –æ—á–µ—Ä–µ–¥–∏ –Ω–∞ –æ—Å–Ω–æ–≤–µ –±–∏–Ω–∞—Ä–Ω–æ–π –∫—É—á–∏, –≤—Ä–µ–º–µ–Ω–Ω–∞—è —Å–ª–æ–∂–Ω–æ—Å—Ç—å –∞–ª–≥–æ—Ä–∏—Ç–º–∞ —Å–æ—Å—Ç–∞–≤–ª—è–µ—Ç **$O(E \log V)$**, –≥–¥–µ:
* $V$ ‚Äî –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –≤–µ—Ä—à–∏–Ω.
* $E$ ‚Äî –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Ä—ë–±–µ—Ä.

---

### –ü—Ä–∏–º–µ—Ä –Ω–∞ Python

```python
import heapq

def dijkstra(graph, start_node):
    """
    –ù–∞—Ö–æ–¥–∏—Ç –∫—Ä–∞—Ç—á–∞–π—à–∏–µ –ø—É—Ç–∏ –æ—Ç start_node –¥–æ –≤—Å–µ—Ö –¥—Ä—É–≥–∏—Ö –≤–µ—Ä—à–∏–Ω
    —Å –ø–æ–º–æ—â—å—é –∞–ª–≥–æ—Ä–∏—Ç–º–∞ –î–µ–π–∫—Å—Ç—Ä—ã.
    """
    # –°–ª–æ–≤–∞—Ä—å –¥–ª—è —Ö—Ä–∞–Ω–µ–Ω–∏—è –∫—Ä–∞—Ç—á–∞–π—à–∏—Ö —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–π
    distances = {node: float('inf') for node in graph}
    distances[start_node] = 0
    
    # –ü—Ä–∏–æ—Ä–∏—Ç–µ—Ç–Ω–∞—è –æ—á–µ—Ä–µ–¥—å: (—Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ, –≤–µ—Ä—à–∏–Ω–∞)
    priority_queue = [(0, start_node)]
    
    while priority_queue:
        # –ò–∑–≤–ª–µ–∫–∞–µ–º –≤–µ—Ä—à–∏–Ω—É —Å –Ω–∞–∏–º–µ–Ω—å—à–∏–º —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ–º
        current_distance, current_node = heapq.heappop(priority_queue)
        
        # –ï—Å–ª–∏ –º—ã –Ω–∞—à–ª–∏ –±–æ–ª–µ–µ –∫–æ—Ä–æ—Ç–∫–∏–π –ø—É—Ç—å —Ä–∞–Ω–µ–µ, –ø—Ä–æ–ø—É—Å–∫–∞–µ–º
        if current_distance > distances[current_node]:
            continue
            
        # –ü—Ä–æ—Å–º–∞—Ç—Ä–∏–≤–∞–µ–º —Å–æ—Å–µ–¥–µ–π —Ç–µ–∫—É—â–µ–π –≤–µ—Ä—à–∏–Ω—ã
        for neighbor, weight in graph[current_node].items():
            distance = current_distance + weight
            
            # –†–µ–ª–∞–∫—Å–∞—Ü–∏—è: –µ—Å–ª–∏ –Ω–∞–π–¥–µ–Ω –±–æ–ª–µ–µ –∫–æ—Ä–æ—Ç–∫–∏–π –ø—É—Ç—å
            if distance < distances[neighbor]:
                distances[neighbor] = distance
                # –î–æ–±–∞–≤–ª—è–µ–º —Å–æ—Å–µ–¥–∞ –≤ –æ—á–µ—Ä–µ–¥—å —Å –æ–±–Ω–æ–≤–ª–µ–Ω–Ω—ã–º —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ–º
                heapq.heappush(priority_queue, (distance, neighbor))
                
    return distances

# –ü—Ä–∏–º–µ—Ä –≤–∑–≤–µ—à–µ–Ω–Ω–æ–≥–æ –≥—Ä–∞—Ñ–∞ –≤ –≤–∏–¥–µ —Å–ø–∏—Å–∫–æ–≤ —Å–º–µ–∂–Ω–æ—Å—Ç–∏
graph_example = {
    'A': {'B': 1, 'C': 4},
    'B': {'A': 1, 'C': 2, 'D': 5},
    'C': {'A': 4, 'B': 2, 'D': 1},
    'D': {'B': 5, 'C': 1}
}

# –ù–∞—Ö–æ–¥–∏–º –∫—Ä–∞—Ç—á–∞–π—à–∏–µ –ø—É—Ç–∏ –æ—Ç –≤–µ—Ä—à–∏–Ω—ã 'A'
shortest_paths = dijkstra(graph_example, 'A')
print(f"–ö—Ä–∞—Ç—á–∞–π—à–∏–µ –ø—É—Ç–∏ –æ—Ç 'A': {shortest_paths}")
# -> –ö—Ä–∞—Ç—á–∞–π—à–∏–µ –ø—É—Ç–∏ –æ—Ç 'A': {'A': 0, 'B': 1, 'C': 3, 'D': 4}
```
